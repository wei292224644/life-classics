# GB 28310—2012 食品添加剂 某种物质

## 范围
本标准规定了食品添加剂某物质的技术要求、试验方法、检验规则、标志、包装、运输、贮存和安全等要求。

## 术语和定义
下列术语和定义适用于本文件：
- **某物质**：指本标准规定的特定食品添加剂。

## 技术要求
### 1. 外观
应为无色透明液体，无异味。

### 2. 理化指标
| 项目 | 指标 |
|------|------|
| 某物质含量 | ≥99.0% |
| 灼烧残渣 | ≤0.05% |
| 乙醇 | ≤0.08% |
| 异丙醇 | ≤0.01% |
| 乙酸乙酯 | ≤0.08% |
| 乙酸异丁酯 | ≤0.1% |

### 3. 卫生指标
应符合GB 2760的规定。

## 试验方法
### 1. 外观检验
目视检查。

### 2. 某物质含量测定
见附录A。

### 3. 灼烧残渣的测定
**方法**：
1. 称取约2 g试样，精确至0.0001 g，置于已在800 ℃±25 ℃灼烧至恒重的坩埚中。
2. 用小火缓缓加热至试样完全炭化，放冷后，加约1.0 mL硫酸使其湿润。
3. 低温加热至硫酸蒸汽除尽后，移入高温炉中，在800 ℃±25 ℃灼烧至恒重。
**结果计算**：
$$ w_2 = \frac{m_1 - m_2}{m} \times 100\% $$
其中：
- $ m_1 $：残渣和空坩埚质量（g）
- $ m_2 $：空坩埚质量（g）
- $ m $：称取的试样质量（g）

### 4. 乙醇、异丙醇、乙酸乙酯、乙酸异丁酯的测定
**方法**：
1. **对照液制备**：
   - 精密吸取乙醇34 μL、异丙醇4.5 μL、乙酸乙酯30 μL、乙酸异丁酯38 μL，置于100 mL容量瓶中，用DMF稀释至刻度。
   - 移取3.0 mL到10 mL顶空瓶中，封盖。
2. **灵敏度溶液制备**：
   - 精密吸取5.0 mL对照液于50 mL容量瓶中，用DMF稀释至刻度。
   - 移取3.0 mL到10 mL顶空瓶中，封盖。
3. **试样液制备**：
   - 精确称取试样约0.1 g到10 mL顶空瓶中，加3.0 mL DMF稀释，封盖，摇匀。
4. **系统适应性要求**：
   - 各个峰理论板数应不小于5000，峰面积的相对标准偏差不得大于10.0%，灵敏度溶液中主峰信噪比应不小于10。
5. **测定**：
   - 分别取对照液和试样液顶空进样，用外标法计算样品中的乙醇、异丙醇、乙酸乙酯和乙酸异丁酯含量。
**结果计算**：
$$ w_i = \frac{A_X \times c_R}{A_R \times c_X} \times 100\% $$
其中：
- $ A_X $：试样液色谱图中所测溶剂峰面积值
- $ c_R $：对照液中对应溶剂浓度（mg/mL）
- $ A_R $：对照液色谱图中所测溶剂峰面积值
- $ c_X $：试样液浓度（mg/mL）

## 检验规则
### 1. 批次划分
以同一批原料、同一工艺、同一规格的产品为一个批次。

### 2. 抽样
按GB/T 6678规定进行抽样。

### 3. 判定规则
所有检验项目均应符合本标准要求，不合格品不得出厂。

## 标志、包装、运输、贮存
### 1. 标志
产品包装上应有产品名称、生产日期、保质期、生产批号、净含量、执行标准号、生产许可证编号等。

### 2. 包装
采用符合食品卫生要求的容器包装，密封保存。

### 3. 运输
运输时应避免阳光直射，保持通风，防止污染。

### 4. 贮存
贮存温度应低于25 ℃，避免高温、潮湿。

## 安全
本产品应符合GB 2760的安全要求。
