# 前言

本标准代替GB5009.211—2014《食品安全国家标准 食品中叶酸的测定》。

本标准与GB5009.211—2014相比，主要变化如下：

——增加了微孔板测定方法；

——修改了精密度要求；

——修改了附录A。

# 食品安全国家标准

# 食品中叶酸的测定

# 1 范围

本标准规定了食品中叶酸的测定方法。

本标准适用于食品中叶酸的测定。

# 2 原理

叶酸是鼠李糖乳杆菌(Lactobacillus rhamnosus)生长所必需的营养素。在一定控制条件下，将鼠李糖乳杆菌菌液接种至含有试样液的培养基中，培养一段时间后测定透光率(或吸光度值)，在一定测定范围内可以根据叶酸含量与透光率(或吸光度值)的标准曲线计算出试样中叶酸的含量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为 $\mathrm{GB / T6682}$ 规定的二级水。

# 3.1 试剂

3.1.1 盐酸（HCl）。  
3.1.2 氢氧化钠 $(\mathrm{NaOH})$ 。  
3.1.3 氯化钠 $(\mathrm{NaCl})$ 。  
3.1.4 十二水合磷酸钠 $(\mathrm{Na}_3\mathrm{PO}_4\cdot 12\mathrm{H}_2\mathrm{O})$ 。  
3.1.5 七水合磷酸氢二钠 $(\mathrm{Na}_2\mathrm{HPO}_4\cdot 7\mathrm{H}_2\mathrm{O})$   
3.1.6 L-抗坏血酸 $\mathrm{(C_6H_8O_6)}$ 。  
3.1.7 甲苯 $(\mathrm{C}_7\mathrm{H}_8)$ 。  
3.1.8 无水乙醇 $(\mathrm{C}_2\mathrm{H}_6\mathrm{O})$ 。  
3.1.9 鸡胰腺冻干粉：含 $\gamma$ -谷胺酰基水解酶。  
3.1.10 木瓜蛋白酶：酶活力 $\geqslant 5\mathrm{U} / \mathrm{mg}$   
3.1.11 $\alpha$ -淀粉酶：酶活力 $\geqslant 1.5 \mathrm{U} / \mathrm{mg}$ 。

# 3.2 试剂配制

3.2.1 磷酸盐缓冲液 $(0.05\mathrm{mol} / \mathrm{L},\mathrm{pH}6.8)$ ：分别称取 $4.35\mathrm{g}$ 十二水合磷酸钠和 $10.39\mathrm{g}$ 七水合磷酸氢二钠，加水溶解并定容至 $1\mathrm{L}$ ，混匀。加入 $2\mathrm{mL}$ 甲苯，室温保存。临用前按约 $5\mathrm{mg / mL}$ 的比例加入L-抗坏血酸作为叶酸保护剂，调节 $\mathrm{pH}$ 至 $6.8\pm 0.1$ 。  
3.2.2 $20\%$ 乙醇溶液 $(2 + 8)$ ：量取 $200~\mathrm{mL}$ 无水乙醇与 $800~\mathrm{mL}$ 水混匀。  
3.2.3 氢氧化钠乙醇溶液 $(0.01\mathrm{mol} / \mathrm{L})$ ：称取 $0.4\mathrm{g}$ 氢氧化钠，用 $20\%$ 乙醇溶液溶解并定容至 $1\mathrm{L}$ ，混匀。  
3.2.4 氢氧化钠溶液 $(1\mathrm{mol} / \mathrm{L})$ ：称取 $40\mathrm{g}$ 氢氧化钠，加水溶解并定容至 $1\mathrm{L}$ ，混匀。

3.2.5 盐酸浸泡液：量取 $100\mathrm{mL}$ 盐酸(浓度 $36\% \sim 38\%$ )与50倍水混合。  
3.2.6 鸡胰腺溶液：称取 $100\mathrm{mg}$ 鸡胰腺冻干粉，加入 $20~\mathrm{mL}$ 磷酸盐缓冲液，摇匀。现用现配。  
3.2.7 蛋白酶-淀粉酶液：分别称取 $200\mathrm{mg}$ 木瓜蛋白酶和 $\alpha$ -淀粉酶，加入 $20~\mathrm{mL}$ 磷酸盐缓冲液研磨至匀浆， $3000\mathrm{r / min}$ 离心 $5\mathrm{min}$ 。现用现配。

# 3.3 培养基

3.3.1 菌种储备用琼脂培养基：按A.1配制。  
3.3.2 叶酸测定培养基：按A.2配制。

注：以上培养基也可用商品化的合成或成品培养基，用前按说明书配制。

# 3.4 标准品

叶酸标准品 $\mathrm{(C_{19}H_{19}N_7O_6}$ ，CAS：59-30-3）：纯度 $\geq 97\%$ ，或经国家认证并授予标准物质证书的标准品。

# 3.5 标准溶液的配制

3.5.1 叶酸标准储备液 $(20.0\mu \mathrm{g / mL})$ ：准确称取 $20.0\mathrm{mg}$ 叶酸标准品，用氢氧化钠乙醇溶液溶解，转移并定容至 $1000~\mathrm{mL}$ 棕色容量瓶中。将溶液转移至棕色玻璃容器中，于 $2^{\circ}C\sim 4^{\circ}C$ 冰箱避光保存，保存期2年。  
3.5.2 叶酸标准中间液 $(0.200\mu \mathrm{g / mL})$ ：准确吸取 $1.00~\mathrm{mL}$ 叶酸标准储备液置于 $100~\mathrm{mL}$ 棕色容量瓶中，用氢氧化钠乙醇溶液稀释并定容至刻度，混匀。将溶液转移至棕色玻璃容器中，于 $2^{\circ}C\sim 4^{\circ}C$ 冰箱避光保存，保存期1年。  
3.5.3 叶酸标准工作液 $(0.200\mathrm{ng / mL})$ ：准确吸取 $1.00~\mathrm{mL}$ 叶酸标准中间液置于 $1000~\mathrm{mL}$ 容量瓶中，用水稀释并定容至刻度，混匀。现用现配。

# 4 仪器和设备

4.1 天平：感量为 $0.1\mathrm{mg}$ 和 $1\mathrm{mg}$   
4.2 恒温培养箱： $36\mathrm{^\circ C}\pm 1\mathrm{^\circ C}$   
4.3 高压灭菌锅。  
4.4 涡旋振荡器。  
4.5 离心机： $3000\mathrm{r / min}$   
4.6 接种环和接种针。   
4.7 pH计：精度为 $\pm 0.1$   
4.8 组织粉碎机和研磨仪。  
4.9 紫外-可见分光光度计。  
4.10 超净工作台。  
4.11 超声波振荡器。  
4.12 酶标仪。  
4.13 离心管。  
4.14 微孔板（无菌）。  
4.15 滤膜 $(0.22\mu \mathrm{m})$ 。  
4.16 容量瓶。

注：所用玻璃器具使用前用盐酸浸泡液或月桂基磺酸钠洗涤剂清洗干净后， $250^{\circ}\mathrm{C}$ 干热 $1\mathrm{h} \sim 2\mathrm{h}$ 。

# 5 菌种的制备与保存

# 5.1 菌种

鼠李糖乳杆菌 Lactobacillus rhamnosus (ATCC 7469) 或等效菌株。

# 5.2 储备菌种的制备

将菌种鼠李糖乳杆菌转接至菌种储备用琼脂培养基中，在 $36^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 恒温培养箱中培养 $20\mathrm{h} \sim 24\mathrm{h}$ ，连续传种2代～3代。取出后放入 $2^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 冰箱作为储备菌株保存。

实验前将储备菌株接种至菌种储备用琼脂培养基中，在 $36\,^{\circ}\mathrm{C} \pm 1\,^{\circ}\mathrm{C}$ 恒温培养箱中培养 $20\,\mathrm{h} \sim 24\,\mathrm{h}$ 以活化菌株，用于接种液的制备。

注：保存2周以上的储备菌种，不能立即用作接种液制备，实验前宜连续传种2代～3代以保证细菌活力。

# 5.3 接种液的制备

实验前一天，取 $2\mathrm{mL}$ 叶酸标准工作液与 $4\mathrm{mL}$ 叶酸测定用培养基混匀，分装至2支试管中，于 $121^{\circ}C$ $(0.10\mathrm{MPa}\sim 0.12\mathrm{MPa})$ 高压灭菌 $15\mathrm{min}$ (或根据培养基标签标识进行灭菌)后即为种子培养液。冷却后用接种环将活化的菌株转种至2支种子培养液中，于 $36^{\circ}C\pm 1^{\circ}C$ 恒温培养箱中培养 $20\mathrm{h}\sim$ $24\mathrm{h}$ 。取出后将种子培养液混悬，无菌操作下吸取 $0.5\mathrm{mL}$ 转种至 $5\mathrm{mL}$ 不加叶酸标准工作液的无菌叶酸测定培养基中，于 $36^{\circ}C\pm 1^{\circ}C$ 再培养 $6\mathrm{h}$ ，以消耗种子培养液中残存在菌株中的多余叶酸，制成接种液。

# 6 分析步骤（所有操作均需避光进行）

# 6.1 试样制备

谷物类、豆类、乳粉等试样需粉碎、研磨、过筛（筛板孔径 $0.3\mathrm{mm}\sim 0.5\mathrm{mm}$ ）；肉、蛋、坚果等用均质器制成食糜；果蔬、半固体食品等试样需匀浆混匀，也可采用冷冻干燥粉碎法混匀；液体试样用前振摇混合。上述制备的试样于 $2^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 冰箱可保存1周。

# 6.2 试样提取

# 6.2.1 直接提取法

测定样品中添加的叶酸含量时，可采用直接提取法。

准确称取固体试样 $0.1\mathrm{g}\sim 2\mathrm{g}$ 或液体试样 $0.5\mathrm{mL}\sim 2\mathrm{mL}$ ，精确至 $0.001\mathrm{g}$ ，转入锥形瓶中，加入 $80~\mathrm{mL}$ 氢氧化钠乙醇溶液，具塞，超声振荡 $0.5\mathrm{h}\sim 4\mathrm{h}$ 至试样完全溶解或分散，转入 $100~\mathrm{mL}$ 容量瓶中，用水定容至刻度。

# 6.2.2 酶解提取法

谷薯类、肉蛋乳类、果蔬菌藻类、豆类及坚果类等食品试样中天然存在的叶酸宜采用酶解提取法。

准确称取适量试样(含 $0.2\mu \mathrm{g}\sim 2\mu \mathrm{g}$ 叶酸)，精确至 $0.001\mathrm{g}$ 。一般谷薯类、肉类、乳类、新鲜果蔬、菌藻类试样 $2\mathrm{g}\sim 5\mathrm{g}$ ；蛋类、豆类、坚果类、内脏、干制试样 $0.2\mathrm{g}\sim 2\mathrm{g}$ ；流质或半流质试样 $5\mathrm{g}\sim 10\mathrm{g}$ 。转入 $100~\mathrm{mL}$ 锥形瓶中，加 $30~\mathrm{mL}$ 磷酸盐缓冲液，振摇 $5\mathrm{min}$ 后，具塞，于 $121^{\circ}C$ (0.10MPa～0.12MPa)高压水解 $15\mathrm{min}$ 。

试样取出后冷却至室温，加入 $1\mathrm{mL}$ 鸡胰腺溶液、 $1\mathrm{mL}$ 蛋白酶-淀粉酶液，混合。加入3滴～5滴甲

苯后，置于 $36\mathrm{^\circ C}\pm 1\mathrm{^\circ C}$ 恒温培养箱内酶解 $16\mathrm{h}\sim 20\mathrm{h}$ 。取出，转入 $100~\mathrm{mL}$ 容量瓶，加水定容至刻度，过滤。

另取一只锥形瓶，不加试样，其他步骤同试样操作，作为酶空白液。

注：以谷物、乳粉等为基质的配方食品如需计量基质本底叶酸含量，可采用酶解法提取。

# 6.3 稀释

根据试样中叶酸含量用水对试样提取液进行适当稀释，使试样稀释液中叶酸含量在 $0.2\mathrm{ng / mL}\sim$ $0.3\mathrm{ng / mL}$ 范围内。

# 6.4 试样测定

# 6.4.1 试管法

# 6.4.1.1 试样和酶空白系列管

取3支试管，分别加入 $1.0\mathrm{mL},2.0\mathrm{mL},3.0\mathrm{mL}$ 试样稀释液 $(V_{x})$ ，补水至 $5.0~\mathrm{mL}$ ，混匀。另取3支试管同法加入酶空白液。每个梯度做2个平行。

# 6.4.1.2 标准系列管

取试管分别加入叶酸标准工作溶液 $0.00\mathrm{mL}, 0.25\mathrm{mL}, 0.50\mathrm{mL}, 1.00\mathrm{mL}, 1.50\mathrm{mL}, 2.00\mathrm{mL}, 2.50\mathrm{mL}, 3.00\mathrm{mL}, 4.00\mathrm{mL}$ 和 $5.00\mathrm{mL}$ ，补水至 $5.00\mathrm{mL}$ ，相当于标准系列管中叶酸含量为 $0.00\mathrm{ng}$ 、 $0.05\mathrm{ng}$ 、 $0.10\mathrm{ng}$ 、 $0.20\mathrm{ng}$ 、 $0.30\mathrm{ng}$ 、 $0.40\mathrm{ng}$ 、 $0.50\mathrm{ng}$ 、 $0.60\mathrm{ng}$ 、 $0.80\mathrm{ng}$ 和 $1.00\mathrm{ng}$ ，混匀。制备2套～3套标准系列管，绘制标准曲线时，以每个标准点平均值计算。

# 6.4.1.3 灭菌

将所有测定系列管、叶酸测定培养基于 $121^{\circ}\mathrm{C}(0.10\mathrm{MPa}\sim 0.12\mathrm{MPa})$ 高压灭菌 $15\mathrm{min}$ （或根据培养基要求进行灭菌）。

# 6.4.1.4 接种和培养

待测定系列管冷却至室温后，在无菌操作条件下，每 $10~\mathrm{mL}$ 叶酸测定培养基加入接种液 $40~\mu \mathrm{L}$ ，混匀，每支测定管中加入接种后的叶酸测定培养基 $5\mathrm{mL}$ ，混匀。置于 $36^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 恒温培养箱中培养 $20\mathrm{h} \sim 40\mathrm{h}$ ，获最大混浊度时终止培养。另准备一支标准0管（含 $0.00\mathrm{ng}$ 叶酸）不接种作为0对照管。

# 6.4.1.5 测定

将培养好的标准系列管、试样和酶空白系列管用漩涡振荡器混匀。用 $1\mathrm{cm}$ 比色杯，于 $540~\mathrm{nm}$ 处，以未接种0对照管调节透光率为 $100\%$ （或吸光度值为0)，依次测定标准系列管、试样和酶空白系列管的透光率(或吸光度值)。如果0对照管出现浑浊，说明可能有杂菌污染，需重做实验。

注：适宜的测定光谱范围为 $540\mathrm{nm}\sim 610\mathrm{nm}$

# 6.4.2 微孔板法

# 6.4.2.1 试样系列管

先将6.3试样稀释液无菌条件下用无菌水相滤膜 $(0.22\mu \mathrm{m})$ 过滤除菌，取3支 $1.5\mathrm{mL}$ 无菌离心管，分别加入 $100~\mu \mathrm{L},200~\mu \mathrm{L},300~\mu \mathrm{L}$ 试样稀释液，补无菌水至 $500~\mu \mathrm{L}$ 。每个梯度做2个平行。

# 6.4.2.2 标准系列管

按6.4.1.2中标准溶液的制备方式，体积按比例缩减至 $1.0\mathrm{mL}$ 后，在无菌条件下过滤除菌至无菌离心管中。制备2套～3套标准系列管，绘制标准曲线时，以每个标准点平均值计算。

# 6.4.2.3 接种和培养

叶酸测定培养基高压灭菌冷却后，每 $10\mathrm{mL}$ 培养基中加入菌种接种液 $40~\mu \mathrm{L}$ ，混匀，吸取 $150~\mu \mathrm{L}$ 加入微孔板中。另吸取 $150~\mu \mathrm{L}$ 标准系列管或试样系列管加入微孔板中，覆膜，混匀，置于 $36^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 恒温培养箱中培养 $32\mathrm{h} \sim 40\mathrm{h}$ 。

# 6.4.2.4 测定

混匀微孔板中培养物，用酶标仪在 $540~\mathrm{nm}$ 处测定吸光度值。

如果0对照孔出现浑浊，说明可能有杂菌污染，需重做实验。

注：适宜的测定光谱范围为 $540\mathrm{nm}\sim 610\mathrm{nm}$

# 7 分析结果表述

# 7.1 标准曲线

以标准系列管叶酸含量为横坐标，每个标准点透光率(或吸光度值)均值为纵坐标，绘制标准曲线。

# 7.2 试样结果计算

从标准曲线计算试样或酶空白系列管中叶酸的相应含量 $(c_{x})$ ，如果3支试样系列管中有2支叶酸含量在 $0.10\mathrm{ng}\sim 0.80\mathrm{ng}$ 范围内，且各管之间折合为每毫升试样提取液中叶酸含量的偏差小于 $10\%$ ，则可继续按式(1)、式(2)、式(3)进行结果计算，否则需重新取样测定。

试样稀释液的叶酸浓度按式(1)计算。

$$
c = \frac {c _ {x}}{V _ {x}} \tag {1}
$$

式中：

$c$ ——试样稀释液中叶酸浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$c_{x}$ ——从标准曲线上查得试样系列管中叶酸含量，单位为纳克 $(\mathrm{ng})$ ；

$V_{x}$ ——制备试样系列管时吸取的试样稀释液体积，单位为毫升 $(\mathrm{mL})$ 。

注：微孔板法吸取 $100\mu \mathrm{L},200\mu \mathrm{L}$ 和 $300~\mu \mathrm{L}$ 时，对应 $V_{x}$ 分别为 $1\mathrm{mL},2\mathrm{mL}$ 和 $3\mathrm{mL}$ 。

采用直接提取法的试样叶酸含量按式(2)计算。

$$
X = \frac {\bar {c} \times V \times f}{m} \times \frac {1 0 0}{1 0 0 0} \tag {2}
$$

式中：

$X$ ——试样中叶酸含量，单位为微克每百克 $(\mu \mathrm{g} / 100\mathrm{g})$ ；

$\overline{c}$ ——试样稀释液叶酸浓度平均值，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V$ ——试样提取液定容体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——试样提取液稀释倍数；

$m$ ——试样质量，单位为克(g)；

$\frac{100}{1000}$ ——由纳克每克 $(\mathrm{ng / g})$ 换算为微克每百克 $(\mu \mathrm{g} / 100\mathrm{g})$ 的系数。

采用酶解提取法的试样叶酸含量按式(3)计算。

$$
X = \frac {\left(\bar {c} \times f - \overline {{c _ {0}}}\right) \times V}{m} \times \frac {1 0 0}{1 0 0 0} \quad \dots \tag {3}
$$

式中：

$c_{0}$ ——酶空白液中叶酸浓度平均值，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

式中 $X, \overline{c}, f, V, m, \frac{100}{1000}$ 的含义同式(2)。

计算结果保留三位有效数字。

注：液体试样叶酸含量也可以微克每百毫升 $(\mu \mathrm{g} / 100\mathrm{mL})$ 为单位。

# 8 精密度

一般食品在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ ；营养素补充剂和强化食品在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 9 其他

试管法：果蔬类试样称样量为 $5\mathrm{g}$ 时，检出限为 $0.2\mu \mathrm{g} / 100\mathrm{g}$ ，定量限为 $0.4\mu \mathrm{g} / 100\mathrm{g}$ ；蛋白质、淀粉含量高的试样称样量为 $5\mathrm{g}$ 时，检出限为 $1.0\mu \mathrm{g} / 100\mathrm{g}$ ，定量限为 $2.0\mu \mathrm{g} / 100\mathrm{g}$ ；营养强化剂和强化食品称样量为 $1\mathrm{g}$ 时，检出限为 $0.5\mu \mathrm{g} / 100\mathrm{g}$ ，定量限为 $1.0\mu \mathrm{g} / 100\mathrm{g}$ 。

微孔板法：添加了叶酸的样品，称样量为 $1.0\mathrm{g}$ ，稀释倍数为1时，检出限为 $0.5\mu \mathrm{g} / 100\mathrm{g}$ ，定量限为 $1.0\mu \mathrm{g} / 100\mathrm{g}$ 。

A.1.1.9 酵母提取物（干粉）：含氮量 $\geqslant 10\%$

A.1.1.10 琼脂。

A.1.1.11 盐酸溶液 $(1\mathrm{mol} / \mathrm{L})$ ：量取 $83.3\mathrm{mL}$ 盐酸（浓度 $36\% \sim 38\%$ ，用水定容至 $1000~\mathrm{mL}$ ，混匀。

# A.1.2 试剂配制

A.1.2.1 甲盐溶液：分别称取 $25\mathrm{g}$ 磷酸氢二钾和 $25\mathrm{g}$ 三水合磷酸二氢钾，加水溶解并定容至 $500~\mathrm{mL}$ ，混匀。加入 $1\mathrm{mL}$ 甲苯，混匀；该溶液于 $2^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 冰箱可保存1年。  
A.1.2.2 乙盐溶液：分别称取 $10\mathrm{g}$ 七水合硫酸镁、 $0.5\mathrm{g}$ 氯化钠、 $0.5\mathrm{g}$ 七水合硫酸亚铁和 $0.5\mathrm{g}$ 一水合硫酸锰，加水溶解并定容至 $500\mathrm{mL}$ 。加5滴 $1\mathrm{mol} / \mathrm{L}$ 盐酸溶液，混匀；该溶液于 $2^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 冰箱可保存1年。  
A.1.2.3 菌种储备用琼脂培养基：按表A.1称量或吸取各试剂，加水至 $100~\mathrm{mL}$ ，混合，沸水浴加热至琼脂完全溶化。趁热用 $1\mathrm{mol} / \mathrm{L}$ 盐酸溶液或 $1\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液调节 $\mathrm{pH}$ 至 $6.8\pm 0.1$ 。尽快分装，根据试管内径粗细加入 $3\mathrm{mL}\sim 5\mathrm{mL}$ ，液面高度不得低于 $2\mathrm{cm}$ 。 $121^{\circ}\mathrm{C}(0.10\mathrm{MPa}\sim 0.12\mathrm{MPa})$ 高压灭菌 $15\mathrm{min}$ 。试管取出后直立放置，待冷却后于 $2^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 冰箱内保存，备用。

# A.2.2 试剂配制

A.2.2.1 氢氧化钠溶液 $(10\mathrm{mol} / \mathrm{L})$ ：称取 $40\mathrm{g}$ 氢氧化钠，用 $100~\mathrm{mL}$ 水溶解。  
A.2.2.2 氢氧化钠溶液 $(1\mathrm{mol} / \mathrm{L})$ ：称取 $4\mathrm{g}$ 氢氧化钠，用 $100~\mathrm{mL}$ 水溶解。  
A.2.2.3 酪蛋白液：称取 $50\mathrm{g}$ 去维生素酪蛋白于 $500~\mathrm{mL}$ 烧杯中，加 $200~\mathrm{mL}$ 盐酸溶液，于 $121^{\circ}C$ $(0.10\mathrm{MPa}\sim 0.12\mathrm{MPa})$ 高压水解 $6\mathrm{h}$ 。将水解物转移至蒸发皿内，在沸水浴上蒸发至膏状。加 $200~\mathrm{mL}$ 水使之溶解后再蒸发至膏状，如此反复3次，以除去盐酸。用 $10\mathrm{mol / L}$ 氢氧化钠调节 $\mathrm{pH}$ 至 $3.5\pm 0.1$ 加 $20\mathrm{g}$ 活性炭，振摇约 $20\mathrm{min}$ ，过滤。重复活性炭处理直至滤液呈淡黄色或无色。滤液加水稀释至 $1000~\mathrm{mL}$ ，加 $1\mathrm{mL}\sim 3\mathrm{mL}$ 甲苯，于 $2^{\circ}C\sim 4^{\circ}C$ 冰箱可保存1年。  
注：每次蒸发时不可蒸干或焦糊，以避免所含营养素破坏。也可直接购买效力相当的酸水解无维生素酪蛋白。  
A.2.2.4 腺嘌呤-鸟嘌呤-尿嘧啶溶液：分别称取硫酸腺嘌呤、盐酸鸟嘌呤以及尿嘧啶各 $0.1\mathrm{g}$ 于 $250~\mathrm{mL}$

烧杯中，加 $75~\mathrm{mL}$ 水和 $2\mathrm{mL}$ 盐酸，加热使其完全溶解后冷却。若有沉淀产生，再加盐酸数滴，加热，如此反复直至冷却后无沉淀产生为止，加水至 $100~\mathrm{mL}$ 。加3滴 $\sim 5$ 滴甲苯，储存于棕色试剂瓶中，于 $2^{\circ}C\sim 4^{\circ}C$ 冰箱可保存1年。

A.2.2.5 黄嘌呤 $\mathrm{(C_5H_4N_4O_2)}$ 溶液：称取 $0.4\mathrm{g}$ 黄嘌呤，加 $10~\mathrm{mL}$ 氨水，加热溶解，加水至 $100~\mathrm{mL}$ 。加3滴～5滴甲苯，储存于棕色试剂瓶中，于 $2^{\circ}C\sim 4^{\circ}C$ 冰箱可保存1年。  
A.2.2.6 乙酸缓冲液 $(1.6\mathrm{mol} / \mathrm{L},\mathrm{pH}4.5)$ ：称取 $63\mathrm{g}$ 三水合乙酸钠，用 $200~\mathrm{mL}$ 水溶解，加大约 $20~\mathrm{mL}$ 冰乙酸调节 $\mathrm{pH}$ 至 $4.5\pm 0.1$ ，混合后，用水稀释至 $500~\mathrm{mL}$ 。  
A.2.2.7 维生素液：称取 $100\mathrm{mg}$ 核黄素用 $400~\mathrm{mL}$ 乙酸缓冲液溶解。取 $25\mathrm{mg}$ 碳酸氢钠溶解于 $500~\mathrm{mL}$ 水中，加入 $2\mathrm{mg}$ 生物素、 $200\mathrm{mg}$ 对氨基苯甲酸、 $400\mathrm{mg}$ 盐酸吡哆醇、 $40\mathrm{mg}$ 盐酸硫胺素、 $80\mathrm{mg}$ 泛酸钙、 $80\mathrm{mg}$ 尼克酸溶解。将上述两种溶液混合，加水至 $1000~\mathrm{mL}$ 。加入3滴～5滴甲苯，储存于棕色试剂瓶中，于 $2^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 冰箱可保存1年。  
A.2.2.8 聚山梨酯-80溶液（吐温-80）：将 $10\mathrm{g}$ 聚山梨酯-80溶于无水乙醇中并稀释至 $100~\mathrm{mL}$ ，于 $2^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 冰箱保存。  
A.2.2.9 还原型谷胱甘肽 $\mathrm{(C_{10}H_{17}N_3O_6S)}$ 溶液：称取 $0.1\mathrm{g}$ 还原型谷胱甘肽，加 $100~\mathrm{mL}$ 水溶解，储于棕色瓶中，现用现配。  
A.2.2.10 磷酸盐缓冲液 $(0.05\mathrm{mol} / \mathrm{L},\mathrm{pH}6.8)$ ：按3.2.1配制。  
A.2.2.11 盐酸溶液 $(1\mathrm{mol} / \mathrm{L})$ ：按A.1.1.11配制。  
A.2.2.12 乙盐溶液：按A.1.2.2配制。

# A.2.3 叶酸测定用培养基

配制 $1000~\mathrm{mL}$ 叶酸测定用培养基，按表A.2吸取液体试剂，混合后加水 $300~\mathrm{mL}$ ，依次加入固体试剂，煮沸搅拌 $2\mathrm{min}$ 。用 $1\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液、 $1\mathrm{mol} / \mathrm{L}$ 盐酸溶液调节 $\mathrm{pH}$ 至 $6.8\pm 0.1$ ；加入乙盐溶液 $20~\mathrm{mL}$ ，用磷酸盐缓冲液补至 $1000~\mathrm{mL}$ 。配制时可根据用量按比例增减，现用现配。
