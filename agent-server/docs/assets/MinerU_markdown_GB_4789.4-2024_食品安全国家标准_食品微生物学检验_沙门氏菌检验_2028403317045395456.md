# 552 多价菌体抗原(O)鉴定

在玻片上划出两个约 $1 \ \mathrm { c m } \times 2 \ \mathrm { c m }$ 的区域 挑取待测菌培养物 各放约一环于玻片上的每一区域上部,在其中一个区域下部加一滴多价菌体( )血清,在另一区域下部加入一滴生理盐水,作为对照。再用无菌的接种环或针将两个区域内的待测菌培养物 分别与血清和生理盐水研成乳状液 将玻片倾斜摇动混合 $1 \ \mathrm { m i n }$ ,并对着黑暗背景进行观察,与对照相比,出现可见的菌体凝集者为阳性反应。 血清不凝集时 将菌株接种在琼脂含量较高 如 $2 \% \sim 3 \%$ 的培养基上培养后再鉴定 如果是由于 抗原的存在而阻止了 血清的凝集反应时 可挑取待测菌培养物在 $1 \ \mathrm { m L }$ 生理盐水中制成浓菌液 在沸水中水浴 $2 0 \ \mathrm { m i n } { \sim } 3 0 \ \mathrm { m i n }$ 冷却后再进行鉴定

注 不同厂商沙门氏菌诊断血清的组成 鉴定操作及结果判断 可能存在差异 使用商品化的沙门氏菌诊断血清进行血清学鉴定时 应遵循其产品说明
