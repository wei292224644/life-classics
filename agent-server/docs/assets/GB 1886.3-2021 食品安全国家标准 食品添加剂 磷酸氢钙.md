# 前言

本标准代替GB1886.3—2016《食品安全国家标准 食品添加剂 磷酸氢钙》。

本标准与GB1886.3—2016相比，主要变化如下：

——修改了范围；  
——增加了无水磷酸氢钙类型；  
将杂质含量指标修改为“以干基计”；  
——将“灼烧失量”修改为“灼烧减量”；  
——将“氟化物（以F计）”修改为“氟(F)”;   
——修改了铅(Pb)的指标值；  
——将氟含量检验方法修改为GB/T5009.18；  
——将铅的检验方法修改为 GB 5009.75—2014 第一法或 GB 5009.12—2017 第二法、第三法、第四法；  
——将砷的检验方法修改为GB5009.76或GB5009.11；  
——将重金属的检验方法修改为GB5009.74。

# 食品安全国家标准

# 食品添加剂 磷酸氢钙

# 1 范围

本标准适用于以氢氧化钙(或碳酸钙、氧化钙)和食品添加剂磷酸(含湿法磷酸)为原料生产的食品添加剂磷酸氢钙。

# 2 分子式和相对分子质量

# 2.1 分子式

二水磷酸氢钙： $\mathrm{CaHPO_4\cdot 2H_2O}$

无水磷酸氢钙： $\mathrm{CaHPO_4}$

# 2.2 相对分子质量

二水磷酸氢钙：172.09（按2018年国际相对原子质量）

无水磷酸氢钙：136.06（按2018年国际相对原子质量）

# 检验方法

警示：本检验方法中使用的部分试剂具有毒性或腐蚀性，操作时应采取适当的安全和防护措施。必要时，应在通风橱中进行。如溅到皮肤上应立即用水冲洗，严重者应立即治疗。

# A.1 一般规定

本标准所用试剂和水在没有注明其他要求时，均指分析纯试剂和GB/T6682中规定的三级水。试验中所用杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602、GB/T603之规定制备。所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 试剂和材料

A.2.1.1 盐酸溶液： $1 + 3$

A.2.1.2 氨水溶液： $1 + 1$

A.2.1.3 硝酸溶液： $1 + 1$

A.2.1.4 草酸铵溶液： $35\mathrm{g / L}$ 。称取 $3.5\mathrm{g}$ 草酸铵 $\left[\left(\mathrm{NH}_{4}\right)\right]_{2}\mathrm{C}_{2}\mathrm{O}_{4}\cdot \mathrm{H}_{2}\mathrm{O}]$ 溶于水，并稀释至 $100~\mathrm{mL}$

A.2.1.5钼酸铵溶液： $60~\mathrm{g / L}$ 。称取 $6\mathrm{g}$ 钼酸铵 $\left[\left(\mathrm{NH}_{4}\right)_{6}\mathrm{Mo}_{7}\mathrm{O}_{24}\cdot 4\mathrm{H}_{2}\mathrm{O}\right]$ 溶于 $50~\mathrm{mL}$ 水中，边搅拌边缓慢加入硝酸溶液至约 $100~\mathrm{mL}$ 。贮存于棕色试剂瓶中。

# A.2.2 鉴别方法

# A.2.2.1 钙离子鉴别

称取约 $0.1\mathrm{g}$ 试样，加入 $5\mathrm{mL}$ 水、 $5\mathrm{mL}$ 盐酸溶液，使试样溶解。边振摇边滴加 $2.5\mathrm{mL}$ 氨水溶液，然后加入 $5\mathrm{mL}$ 草酸铵溶液，产生白色沉淀。

# A.2.2.2 磷酸根离子的鉴别

称取约 $0.1\mathrm{g}$ 试样，滴加硝酸溶液至试样溶解后再过量 $1\mathrm{mL}$ ，加热至 $40^{\circ}\mathrm{C} \sim 50^{\circ}\mathrm{C}$ ，加入 $10\mathrm{mL}$ 钼酸铵溶液，生成黄色沉淀。

# A.3 磷酸氢钙含量的测定

# A.3.1 方法提要

在试验溶液中, 三乙醇胺作为掩蔽剂, 在酸性条件下滴加乙二胺四乙酸二钠标准滴定溶液近终点,用氢氧化钠溶液调试验溶液的 $\mathrm{pH}$ 于羟基萘酚蓝指示剂显色 $\mathrm{pH}$ 范围内, 继续滴定剩余的钙。

# A.3.2 试剂和材料

A.3.2.1 三乙醇胺。  
A.3.2.2 盐酸溶液： $1 + 1$   
A.3.2.3 氢氧化钠溶液： $450~\mathrm{g / L}$

A.3.2.4 乙二胺四乙酸二钠标准滴定溶液： $c(\mathrm{EDTA}) = 0.05\mathrm{mol / L}$   
A.3.2.5 羟基萘酚蓝指示剂。

# A.3.3 仪器和设备

磁性搅拌器。

# A.3.4 分析步骤

# A.3.4.1 试验溶液的制备

称取 $2.5\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ ，置于 $100\mathrm{mL}$ 烧杯中，加 $20\mathrm{mL}$ 盐酸溶液溶解，完全转移至 $250\mathrm{mL}$ 容量瓶中，加水至刻度，摇匀。此溶液为试验溶液。

# A.3.4.2 测定

准确移取 $25~\mathrm{mL}$ 试验溶液于 $250~\mathrm{mL}$ 锥形瓶中，加 $100~\mathrm{mL}$ 水、 $0.5~\mathrm{mL}$ 三乙醇胺，加入 $0.3\mathrm{g}$ 羟基萘酚蓝指示剂，放于磁性搅拌器上搅拌均匀，用滴定管以正常速度滴加乙二胺四乙酸二钠标准滴定溶液约 $23~\mathrm{mL}$ ，滴加氢氧化钠溶液由红色变至亮蓝色，最后变为紫色，再过量 $0.5~\mathrm{mL}$ 。继续用乙二胺四乙酸二钠标准滴定溶液滴定至溶液变为蓝色，保持 $60~\mathrm{s}$ 不变。同时做空白试验。

# A.3.5 结果计算

磷酸氢钙(以 $\mathrm{CaHPO_4\cdot 2H_2O}$ 计或以 $\mathrm{CaHPO_4}$ 计)的质量分数 $\varpi_{1}$ 按式(A.1)计算。

$$
w _ {1} = \frac {\left[ \left(V - V _ {0}\right) / 1000 \right] \times c \times M \times 250}{m _ {1} \times 25} \times 100 \% \dots \tag{A.1}
$$

式中：

$V$ ——滴定试样溶液消耗的乙二胺四乙酸二钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{0}$ ——滴定空白试验溶液消耗的乙二胺四乙酸二钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

1000 ——换算因子；

$c$ ——乙二胺四乙酸二钠标准滴定溶液浓度的准确数值，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M$ ——二水磷酸氢钙 $(\mathrm{CaHPO_4}\cdot 2\mathrm{H_2O})$ 、磷酸氢钙 $(\mathrm{CaHPO_4})$ 的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})[M(\mathrm{CaHPO_4}\cdot 2\mathrm{H_2O}) = 172.09,M(\mathrm{CaHPO_4}) = 136.06]$

250 ——容量瓶的容积，单位为毫升 $(\mathrm{mL})$ ；

$m_{1}$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

25 ——移取试验溶液的体积，单位为毫升 $(\mathrm{mL})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果绝对差值不大于 $0.5\%$ 。

# A.4 灼烧减量的测定

# A.4.1 仪器和设备

A.4.1.1 高温炉：控温范围为 $800^{\circ}\mathrm{C} \sim 825^{\circ}\mathrm{C}$

A.4.1.2 瓷坩埚： $30~\mathrm{mL}$

# A.4.2 分析步骤

称取 $3\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ ，置于已在 $800^{\circ}\mathrm{C} \sim 825^{\circ}\mathrm{C}$ 灼烧至质量恒定的瓷坩埚中，于 $800^{\circ}\mathrm{C} \sim$

$825~\mathrm{^\circ C}$ 灼烧至质量恒定。

# A.4.3 结果计算

灼烧减量的质量分数 $\omega_{2}$ 按式(A.2)计算。

$$
w _ {2} = \frac {m _ {2} - m _ {3}}{m _ {4}} \times 100 \% \quad \dots \tag{A.2}
$$

式中：

$m_{2}$ ——灼烧前坩埚和试样的质量，单位为克 $(\mathrm{g})$ ；

$m_{3}$ ——灼烧后坩埚和试样的质量，单位为克 $(\mathrm{g})$ ；

$m_{4}$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果绝对差值不大于 $0.2\%$ 。

# A.5 盐酸不溶物的测定

# A.5.1 试剂和材料

盐酸溶液： $1 + 1$

# A.5.2 仪器和设备

A.5.2.1 玻璃砂坩埚：滤板孔径 $5\mu \mathrm{m}\sim 15\mu \mathrm{m}$   
A.5.2.2 电热恒温干燥箱：控温范围为 $105\,^{\circ}\mathrm{C} \pm 2\,^{\circ}\mathrm{C}$ 。

# A.5.3 分析步骤

称取试样 $10\mathrm{g}$ ，精确至 $0.01\mathrm{g}$ ，置于 $250~\mathrm{mL}$ 烧杯中，加入 $40~\mathrm{mL}$ 盐酸溶液和 $60~\mathrm{mL}$ 水，加热溶解，加水至 $100~\mathrm{mL}$ ，用 $105^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 下已质量恒定的玻璃砂坩埚过滤，用水洗涤至无氯离子，于 $105^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 的电热恒温干燥箱中干燥至质量恒定。

# A.5.4 结果计算

盐酸不溶物的质量分数 $w_{3}$ 按式(A.3)计算。

$$
w _ {3} = \frac {m _ {5} - m _ {6}}{m _ {7} \times \left(1 - w _ {2}\right)} \times 100 \% \quad \dots \tag{A.3}
$$

式中：

$m_{5}$ ——盐酸不溶物和玻璃砂坩埚的质量，单位为克 $(\mathrm{g})$ ；

$m_{6}$ ——玻璃砂坩埚的质量，单位为克 $(\mathrm{g})$ ；

$m_{7}$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

$\varpi_{2}$ ——按A.4测定所得灼烧失量的质量分数， $\%$

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果绝对差值不大于 $0.01\%$ 。

# A.6 铅(Pb)的测定

称取适量经A.4灼烧后的试样，按GB5009.75—2014第一法或GB5009.12—2017第二法、第三法、第四法规定的方法进行测定，试验中所用水为GB/T6682规定的二级水。

# A.7 砷（As）的测定

称取适量经A.4灼烧后的试样，滴加盐酸溶液 $(1 + 1)$ 至试样溶解再过量 $2\mathrm{mL}$ ，然后按GB5009.76或GB5009.11规定的方法进行测定，试验步骤中用盐酸(或盐酸溶液)代替硫酸(或硫酸溶液)，试验中所用水为GB/T6682规定的二级水。

# A.8 重金属（以 $\mathbf{Pb}$ 计）的测定

称取经A.4灼烧后的试样 $2\mathrm{g}\pm 0.01\mathrm{g}$ ，加 $5\mathrm{mL}$ 盐酸溶液 $(1 + 1)$ 、 $20~\mathrm{mL}$ 水，加热至溶解，冷却后滴加氨水溶液 $(1 + 5)$ 至白色沉淀出现，再加入少量盐酸溶液 $(1 + 1)$ 至沉淀消失，按GB5009.74规定的方法，加入 $5\mathrm{mLpH}3.5$ 的乙酸盐缓冲液，然后进行测定。试验中所用水为GB/T6682规定的二级水。
