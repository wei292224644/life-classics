# 前言

本文件按照GB/T1.1—2020《标准化工作导则第1部分：标准化文件的结构和起草规则》的规定起草。

本文件代替GB29696—2013《牛奶中阿维菌素类药物多残留的测定 高效液相色谱法》及GB31659.4—2022《食品安全国家标准 奶及奶粉中阿维菌素类药物残留量的测定 液相色谱串联质谱法》。

与GB29696--2013相比，除结构调整和编辑性改动外，主要技术内容变化如下：

a）更改了标准适用范围，扩大至牛奶、羊奶及奶粉等；  
b）更改了部分测定步骤；  
c) 降低了方法的检出限和定量限, 本方法在牛羊奶中的检出限为 $0.2\mu \mathrm{g} / \mathrm{kg}$ , 定量限为 $0.5\mu \mathrm{g} / \mathrm{kg}$ ; 在奶粉中的检出限为 $2\mu \mathrm{g} / \mathrm{kg}$ , 定量限为 $5\mu \mathrm{g} / \mathrm{kg}$ 。

与GB31659.4-2022相比，除结构调整和编辑性改动外，主要增加了高效液相色谱法测定步骤。

# 方法一 高效液相色谱法

# 1 范围

本文件规定了奶及乳粉中乙酰氨基阿维菌素、阿维菌素、多拉菌素和伊维菌素残留量检测的制样和高效液相色谱测定方法。

本文件适用于牛奶、羊奶及其乳粉中乙酰氨基阿维菌素 $\mathrm{B}_{1\mathrm{a}}$ 、阿维菌素 $\mathrm{B}_{1\mathrm{a}}$ 、多拉菌素和22，23-二氢阿维菌素 $\mathrm{B}_{1\mathrm{a}}$ 单个或多个药物残留量的检测，其他类型奶及乳粉的检测可参照执行。

下列文件中的内容通过文中的规范性引用而构成本文件必不可少的条款。其中，注日期的引用文件，仅该日期对应的版本适用于本文件；不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

# 3 术语和定义

# 4 原理

试样中残留的阿维菌素类药物用乙醇-乙腈溶液提取，C、固相萃取柱净化，三氟乙酸酐和 $N$ -甲基咪唑衍生化。高效液相色谱-荧光检测法测定，外标法定量。

# 5 试剂与材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水。

# 5.1 试剂

5.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
5.1.2 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
5.1.3 无水乙醇 $\mathrm{(C_2H_5OH)}$ ：色谱纯。  
5.1.4 异辛烷 $(\mathrm{C_8H_{18}})$   
5.1.5 三乙胺 $(\mathrm{C_6H_{15}N})$   
5.1.6 三氟乙酸酐 $(\mathrm{C_4F_6O_3})$ 。  
5.1.7 $N$ -甲基咪唑 $(\mathrm{C_4H_6N_2})$

# 5.2 标准品

阿维菌素、伊维菌素、多拉菌素、乙酰氨基阿维菌素含量均 $\geq 94\%$ 。具体见附录A。

# 5.3 溶液配制

# 5.3.3 衍生化试剂

5.3.1 $20\%$ 乙醇乙腈溶液：取无水乙醇 $20~\mathrm{mL}$ 用乙腈稀释至 $100~\mathrm{mL}$   
5.3.2 三乙胺乙腈水溶液；取乙腈 $30~\mathrm{mL}$ 、水 $70~\mathrm{mL}$ 和三乙胺 $20~\mu \mathrm{L}$ ，混匀。

a）A液：取 $N$ -甲基咪唑 $5\mathrm{mL}$ 、乙腈 $5\mathrm{mL}$ ，混匀。现用现配。  
b）B液：取三氟乙酸酐 $5\mathrm{mL}$ 、乙腈 $10\mathrm{mL}$ ，混匀。现用现配。

# 5.4 标准溶液配制

5.4.1 阿维菌素类药物混合标准储备液：分别称取乙酰氨基阿维菌素、阿维菌素、多拉菌素和伊维菌素标准品各约 $10\mathrm{mg}$ ，精密称定，用乙腈溶解并定容至 $10\mathrm{mL}$ 容量瓶中，配制成浓度为 $1\mathrm{mg/mL}$ 的混合标准储备液， $-18^{\circ}\mathrm{C}$ 保存，有效期6个月。  
5.4.2 阿维菌素类药物混合标准中间液：准确量取混合标准储备液 $0.25\mathrm{mL}$ 于 $25\mathrm{mL}$ 容量瓶中，用乙腈稀释至刻度，配制成浓度为 $10~\mu \mathrm{g / mL}$ 的混合标准工作液。 $-18^{\circ}C$ 保存，有效期3个月。  
5.4.3 阿维菌素类药物混合标准工作液 $(1\mu \mathrm{g} / \mathrm{mL})$ : 准确量取阿维菌素类药物混合标准中间液 $2.5\mathrm{mL}$ 于 $25\mathrm{mL}$ 容量瓶中, 用乙腈稀释至刻度, 配制成浓度为 $1\mu \mathrm{g} / \mathrm{mL}$ 的混合标准工作液。 $-18^{\circ}\mathrm{C}$ 保存, 有效期3个月。  
5.4.4 阿维菌素类药物混合标准工作液 $(100\mathrm{ng / mL})$ ：准确量取阿维菌素类药物混合标准中间液 $0.25\mathrm{mL}$ 于 $25~\mathrm{mL}$ 容量瓶中，用乙腈稀释至刻度，配制成浓度为 $100\mathrm{ng / mL}$ 的混合标准工作液。 $-18^{\circ}C$ 保存，有效期3个月。

# 7 试样的制备与保存

# 7.1 试样的制备

取适量新鲜或解冻的空白或供试牛奶或羊奶及奶粉，混合均匀。

a）取混匀后的供试样品，作为供试试样；  
b）取混匀后的空白样品，作为空白试样；  
c）取混匀后的空白样品，添加适宜浓度的标准工作液，作为空白添加试样。

# 7.2 试样的保存

7.2.1 牛奶和羊奶：冷藏避光或 $-18^{\circ}C$ 以下保存。  
7.2.2 奶粉：阴凉干燥处保存。

# 8 测定步骤

# 8.1 提取

称取牛羊奶试料 $(5 \pm 0.05) \mathrm{g}$ 、奶粉 $(0.5 \pm 0.05) \mathrm{g}$ 于 $50 \mathrm{~mL}$ 离心管中，奶粉样品加 $4 \mathrm{~g}$ 水溶解，充分混匀。奶及奶粉试样加 $20\%$ 乙醇乙腈溶液 $8 \mathrm{~mL}$ ，涡旋混匀， $250 \mathrm{r} / \mathrm{min}$ 振荡 $5 \mathrm{~min}$ ， $5000 \mathrm{r} / \mathrm{min}$ 离心 $10 \mathrm{~min}$ 。收集上清液于另一 $50 \mathrm{~mL}$ 离心管中。残渣再重复提取1次。合并2次上清液，加水 $15 \mathrm{~mL}$ 和三乙胺 $50 \mu \mathrm{L}$ ，混匀，备用。

# 8.2 净化

依次用乙腈 $5\mathrm{mL}$ 、三乙胺乙腈水溶液 $5\mathrm{mL}$ ，活化 $\mathrm{C}_{18}$ 固相萃取柱。将备用液过柱，抽干。加异辛烷 $3\mathrm{mL}$ 洗涤，抽干。用乙腈 $5\mathrm{mL}$ 洗脱。收集洗脱液于 $10~\mathrm{mL}$ 试管中， $50^{\circ}C$ 下用氮气吹干。备用。

# 8.3 衍生化

向试管中依次加入衍生化试剂A液 $100\mu \mathrm{L}$ 和衍生化试剂B液 $150~\mu \mathrm{L}$ ，密闭，涡旋 $10\mathrm{s}$ ，依次加冰醋酸、三乙胺各 $50~\mu \mathrm{L}$ ，涡旋 $10\mathrm{s},65^{\circ}C$ 避光密闭反应 $15\mathrm{min}$ ，取出后放置至室温，加 $650~\mu \mathrm{L}$ 甲醇，混匀。经 $0.45\mu \mathrm{m}$ 微孔滤膜过滤，供高效液相色谱检测。

# 8.4 标准曲线的制备

精密量取 $100\mathrm{ng / mL}$ 阿维菌素类药物混合标准工作液 $0.15\mathrm{mL},0.5\mathrm{mL}$ 于 $10~\mathrm{mL}$ 容量瓶中，用乙腈稀释至刻度；精密量取 $1\mu \mathrm{g} / \mathrm{mL}$ 阿维菌素类药物混合标准工作液 $0.5\mathrm{mL},1.0\mathrm{mL},2.5\mathrm{mL},5.0\mathrm{mL}$ 至 $10~\mathrm{mL}$ 容量瓶中，用乙腈稀释至刻度。配制成浓度分别为 $1.5\mathrm{ng / mL},5.0\mathrm{ng / mL},50.0\mathrm{ng / mL},100.0\mathrm{ng / mL},$ $250.0\mathrm{ng / mL},500.0\mathrm{ng / mL}$ 的乙酰氨基阿维菌素、阿维菌素、多拉菌素和伊维菌素混合标准溶液。分别取上述6个浓度及 $1\mu \mathrm{g / mL}$ 的标准工作液各 $1.0\mathrm{mL}$ ， $10\mathrm{mL}$ 试管中，于 $50^{\circ}C$ 氮气吹干，按8.3衍生化步骤处理后供高效液相色谱测定。以测得峰面积为纵坐标、对应的标准溶液浓度为横坐标，绘制标准曲线。求回归方程和相关系数。

# 8.5 测定

# 8.5.1 色谱条件

a） 色谱柱： $C_{18}(250\mathrm{mm}\times 4.6\mathrm{mm},3\mathrm{mm})$ ，或相当者  
b）流动相：乙腈+水（90-10V/V）；  
c) 流速: $1.8 \mathrm{~mL} / \mathrm{min}$ ;   
d) 检测波长: 激发波长为 $363 \mathrm{~nm}$ , 发射波长为 $475 \mathrm{~nm}$ ;  
e) 柱温：40℃  
f） 进样量： $40 - \mu \mathrm{I}$

# 8.5.2 测定法

取试料溶液和相应的标准溶液，做单点或多点校准，以相应药物的保留时间定性。被测试样中乙酰氨基阿维菌素、阿维菌素、多拉菌素和伊维菌素色谱峰保留时间与相应标准溶液色谱峰的保留时间相比，相差应在 $\pm 0.1\mathrm{min}$ 内。按外标法以峰面积计算。标准溶液及试料溶液中1种阿维菌素类药物响应值均应在仪器检测的线性范围之内。在上述色谱条件下，标准溶液高效液相色谱图见附录B的图B.1。

# 8.6 空白试验

取空白试样，除不加标准溶液外，采用完全相同的测定步骤进行平行操作。

# 9 结果计算和表述

试样中阿维菌素类药物的残留量按标准曲线或公式(1)计算

$$
X = \frac {A \times C _ {s} \times V}{A _ {s} \times m} \dots \tag {1}
$$

式中：

$X$ ——试样中相应阿维菌素类药物残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$A$ ——试样溶液中阿维菌素类药物的色谱峰面积；

$C_{\mathrm{s}}$ ——标准溶液中阿维菌素类药物浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$V$ ——试料最终定容体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$A_{s}$ ——标准溶液中阿维菌素类药物的色谱峰面积；

$m$ ——供试试料质量的数值，单位为克 $(\mathrm{g})$ 。

# 10 检测方法灵敏度、正确度和精密度

# 10.1 灵敏度

本方法在牛羊奶中的检出限为 $0.2\mu \mathrm{g / kg}$ ，定量限为 $0.5\mu \mathrm{g / kg}$ ；在奶粉中的检出限为 $2\mu \mathrm{g / kg}$ ，定量限为 $4\mu \mathrm{g / kg}$ 。

# 10.2 正确度

本方法在牛羊奶 $0.5\mu \mathrm{g} / \mathrm{kg}\sim 50~\mu \mathrm{g} / \mathrm{kg}$ 添加浓度的回收率为 $70\% \sim 120\%$ ；在奶粉 $4\mu \mathrm{g} / \mathrm{kg}\sim 500~\mu \mathrm{g} / \mathrm{kg}$ 添加浓度的回收率为 $70\% \sim 120\%$ 。

# 10.3 精密度

本方法的批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 20\%$ 。

# 方法二 液相色谱-串联质谱法

# 11 原理

# 15 测定步骤

# 15.1 提取

称取牛奶或羊奶 $(2\pm 0.05)\mathrm{g}$ 或乳粉 $(0.3\pm 0.05)\mathrm{g}$ 于 $50~\mathrm{mL}$ 离心管中，加乙腈 $3\mathrm{mL}$ ，涡旋后 $200~\mathrm{r / min}$ 水平振荡 $5\mathrm{min}$ ， $8000~\mathrm{r / min}$ 离心 $8\mathrm{min}$ ，转移1，清液至另一 $50~\mathrm{mL}$ 离心管中，加水 $10~\mathrm{mL}$ ，三乙胺 $25~\mu \mathrm{L}$ ，混匀，备用。

# 15.2 净化与浓缩

$\mathrm{C_{18}}$ 固相萃取柱依次用乙腈 $5 \mathrm{~mL}$ 和淋洗溶液 $5 \mathrm{~mL}$ 活化，取备用液过柱，用淋洗溶液 $5 \mathrm{~mL}$ 淋洗，抽干；再用异辛烷 $5 \mathrm{~mL}$ 淋洗、抽干；用乙腈 $5 \mathrm{~mL}$ 洗脱，收集洗脱液于 $10 \mathrm{~mL}$ 试管中，于 $50^{\circ} \mathrm{C}$ 下氮气吹干。残余物中加入 $50 \%$ 乙腈溶液 $1.0 \mathrm{~mL}$ ，充分溶解。过微孔滤膜后供液相色谱-串联质谱仪测定。

# 15.3 基质匹配标准曲线的制备

精密量取阿维菌素类标准工作液适量，用 $50\%$ 乙腈水溶液稀释成含阿维菌素 $\mathrm{B}_{1a}$ 、22，23-二氢阿维菌素 $\mathrm{B_{la}}$ 、多拉菌素和乙酰氨基阿维菌素 $\mathrm{B}_{\mathrm{a}}$ 浓度分别为 $0.5\mathrm{ng / mL},1\mathrm{ng / mL},2\mathrm{ng / mL},5\mathrm{ng / mL},10\mathrm{ng / mL},$ $20~\mathrm{ng / mL},50~\mathrm{ng / mL},100~\mathrm{ng / mL}$ 和 $400~\mathrm{ng / mL}$ 的系列标准工作液（牛奶或牛奶用 $0.5\mathrm{ng / mL},1\mathrm{ng / mL},$ $2\mathrm{ng / mL},5\mathrm{ng / mL},10\mathrm{ng / mL},20\mathrm{ng / mL}$ 和 $50~\mathrm{ng / mL}$ 浓度点，奶粉用 $2\mathrm{ng / mL},5\mathrm{ng / mL},10\mathrm{ng / mL},$ $20~\mathrm{ng / mL},50~\mathrm{ng / mL},100~\mathrm{ng / mL}$ 和 $400~\mathrm{ng / mL}$ 浓度点），从中各取 $1.0~\mathrm{mL}$ ，分别加入空白试料经提取、净化和吹干后的残余物中，充分溶解，作为基质匹配标准溶液；过微孔滤膜后上机测定。以特征离子质量色谱峰面积为纵坐标、基质匹配标准溶液浓度为横坐标，绘制标准曲线。

# 15.4 测定

# 15.4.1 液相色谱参考条件

b）流动相：A为 $0.1\%$ 甲酸溶液，B为 $0.1\%$ 甲酸乙腈溶液，流动相梯度： $0\mathrm{min}\sim 1\mathrm{min}$ 保持 $70\%$ B； $1\mathrm{min}\sim 3\mathrm{min},70\%$ B线性变化到 $100\%$ B, $3\mathrm{min}\sim 4\mathrm{min}$ 保持 $100\%$ B, $4\mathrm{min}\sim 6\mathrm{min}$ 保持 $70\%$ B。

# 15.4.3 定性测定

在相同测试条件下，试料溶液中阿维菌素类药物与基质匹配标准溶液中药物的保留时间偏差在 $\pm 0.1\mathrm{min}$ 以内；且检测到的相对离子丰度，应与浓度相当的基质匹配标准溶液相对离子丰度一致，其允许偏差为 $\pm 40\%$ 。

# 15.4.4 定量测定

取试料溶液和基质匹配标准溶液，做单点或多点校准，外标法计算。基质匹配标准溶液及试料溶液中目标物响应值均应在仪器检测的线性范围内。标准溶液特征离子质量色谱图见附录B中图B.2。

# 15.5 空白试验

取空白试样，除不添加药物外，采用完全相同的测定步骤进行平行操作。

# 16 结果计算和表述

试样中阿维菌素类药物的残留量按标准曲线或公式(2)计算。

$$
X = \frac {C \times A \times V \times 1 0 0 0}{A _ {\mathrm {s}} \times m \times 1 0 0 0} \tag {2}
$$

式中：

$X$ ——供试试样中阿维菌素类药物残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

C 基质匹配标准溶液中阿维菌素类药物浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$

$A$ ——试料溶液中阿维菌素类药物的峰面积；

$V$ ——溶解残余物所用溶液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$A_{s}$ ——基质匹配标准溶液中阿维菌素类药物的峰面积；

$m$ ——供试试料质量的数值，单位为克(g)；

1000——换算系数。

# 17 检测方法的灵敏度、正确度和精密度

# 17.1 灵敏度

本方法对牛奶或羊奶的检出限为 $0.2\mu \mathrm{g / kg}$ ，定量限为 $0.5\mu \mathrm{g / kg}$ ；对奶粉的检出限为 $1\mu \mathrm{g / kg}$ ，定量

限为 $4\mu \mathrm{g / kg}$

# 17.2 正确度

本方法牛奶中阿维菌素 $\mathrm{B_{1a}}$ 在 $0.5\mu \mathrm{g} / \mathrm{kg}\sim 5\mu \mathrm{g} / \mathrm{kg},22,23-$ 二氢阿维菌素 $\mathrm{B_{1a}}$ 在 $0.5\mu \mathrm{g} / \mathrm{kg}\sim 20\mu \mathrm{g} / \mathrm{kg}$ 、多拉菌素在 $0.5\mu \mathrm{g} / \mathrm{kg}\sim 30\mu \mathrm{g} / \mathrm{kg}$ 、乙酰氨基阿维菌素 $\mathrm{B_{1a}}$ 在 $0.5\mu \mathrm{g} / \mathrm{kg}\sim 40\mu \mathrm{g} / \mathrm{kg}$ 添加浓度上，羊奶中阿维菌素类药物在 $0.5\mu \mathrm{g} / \mathrm{kg}\sim 5\mu \mathrm{g} / \mathrm{kg}$ 添加浓度水平上，以及奶粉中阿维菌素 $\mathrm{B_{1a}}$ 在 $4\mu \mathrm{g} / \mathrm{kg}\sim 40\mu \mathrm{g} / \mathrm{kg},22,$ $23-$ 二氢阿维菌素 $\mathrm{B_{1a}}$ 在 $4\mu \mathrm{g} / \mathrm{kg}\sim 160\mu \mathrm{g} / \mathrm{kg}$ 、多拉菌素在 $4\mu \mathrm{g} / \mathrm{kg}\sim 240\mu \mathrm{g} / \mathrm{kg}$ 、乙酰氨基阿维菌素 $\mathrm{B_{1a}}$ 在 $4\mu \mathrm{g} / \mathrm{kg}\sim 320\mu \mathrm{g} / \mathrm{kg}$ 添加浓度上的回收率均为 $60\% \sim 120\%$ 。

# 17.3 精密度

本方法批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 20\%$
