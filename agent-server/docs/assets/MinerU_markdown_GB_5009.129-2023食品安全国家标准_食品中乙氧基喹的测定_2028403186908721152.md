# 食品安全国家标准

# 食品中乙氧基喹的测定

2023-09-06 发布

2024-03-06 实施

# 前言

本标准代替GB/T5009.129—2003《水果中乙氧基喹残留量的测定》。

本标准与GB/T5009.129—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中乙氧基喹的测定”；

——增加了液相色谱法；

——修改了第二法气相色谱法的试样处理方法、色谱参考条件和定量方法。

# 食品安全国家标准

# 食品中乙氧基喹的测定

# 1 范围

本标准规定了食品中乙氧基喹的液相色谱测定方法和气相色谱测定方法。

本标准适用于新鲜水果中乙氧基喹的测定。

# 第一法 液相色谱法

# 2 原理

试样中的乙氧基喹在碱性条件下经正己烷提取，氮吹至干后，含抗坏血酸的乙腈溶液复溶，液相色谱仪测定，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 氢氧化钠(NaOH)。

3.1.2 抗坏血酸 $(\mathrm{C_6H_8O_6})$ 。

3.1.3 乙腈 $\mathrm{(C_2H_3N)}$ ：色谱纯。

3.1.4 正己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{14}\right)$ 。

# 3.2 试剂配制

3.2.1 氢氧化钠溶液 $(0.1\mathrm{mol} / \mathrm{L})$ ：称取 $4\mathrm{g}$ 氢氧化钠，溶于水并定容至 $1000~\mathrm{mL}$ 。

3.2.2 含抗坏血酸的乙腈溶液：称取抗坏血酸 $0.1\mathrm{g}$ ，溶于 $100~\mathrm{mL}$ 乙腈中，充分振摇，过滤，取滤液，临用现制。

# 3.3 标准品

乙氧基喹标准品 $\mathrm{(C_{14}H_{19}NO}$ ，CAS号：91-53-2）：纯度 $\geq 99\%$ ，或经国家认证并授予标准物质证书的标准品。

# 3.4 标准溶液配制

3.4.1 乙氧基喹标准储备液 $(1.00\mathrm{mg / mL})$ : 称取乙氧基喹标准品 $100\mathrm{mg}$ (精确至 $0.1\mathrm{mg}$ ) 于 $100\mathrm{mL}$ 容量瓶中, 用含抗坏血酸的乙腈溶液溶解并定容至刻度, 混匀。将溶液转移至棕色玻璃容器中, 于 $-18^{\circ}C$ 下避光保存, 有效期为 3 个月。

3.4.2 乙氧基喹标准中间液 $(10.0\mu \mathrm{g / mL})$ ：吸取标准储备液 $(1.00\mathrm{mg / mL})1.00\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶

中,用含抗坏血酸的乙腈溶液定容至刻度,混匀。将溶液转移至棕色玻璃容器中,于 $-18^{\circ} \mathrm{C}$ 下避光保存,有效期为 1 个月。

3.4.3 乙氧基喹标准使用液 $(1.00\mu \mathrm{g} / \mathrm{mL})$ ：吸取标准中间液 $(10.0\mu \mathrm{g} / \mathrm{mL})1.00\mathrm{mL}$ 于 $10~\mathrm{mL}$ 容量瓶中，用含抗坏血酸的乙腈溶液定容至刻度，混匀。临用现配。

3.4.4 乙氧基喹标准系列工作液：分别吸取标准使用液 $(1.00\mu \mathrm{g / mL})$ 40.0 $\mu \mathrm{L}, 100\mu \mathrm{L}, 200\mu \mathrm{L}, 500\mu \mathrm{L}, 1000\mu \mathrm{L}$ 于 $10\mathrm{mL}$ 容量瓶中，用含抗坏血酸的乙腈溶液定容至刻度，混匀。乙氧基喹标准系列工作液的浓度分别为 $4.00\mathrm{ng / mL}, 10.0\mathrm{ng / mL}, 20.0\mathrm{ng / mL}, 50.0\mathrm{ng / mL}, 100\mathrm{ng / mL}$ 。临用现配。

# 4 仪器和设备

4.1 液相色谱仪：配有荧光检测器。

4.2 组织捣碎机。

4.3 分析天平：感量分别为 $0.01\mathrm{g}$ 和 $0.1\mathrm{mg}$ 

4.4 涡旋混合器。

4.5 振荡器。

4.6 氮气浓缩装置。

4.7 离心机。

# 5 分析步骤

# 5.1 试样制备

取代表性试样可食部分 $500\mathrm{g}$ ，加入抗坏血酸 $10\mathrm{g}$ ，用组织捣碎机粉碎混匀加工成浆状，装入洁净的容器中，密封。制备好的试样于 $-18^{\circ}\mathrm{C}$ 冷冻保存，尽快测定。

注：当采用GB2763进行判定时，测定部位按GB2763的规定执行。

# 5.2 试样提取

称取 $5\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ）试样于 $50~\mathrm{mL}$ 离心管中，加入氢氧化钠溶液 $(0.1\mathrm{mol} / \mathrm{L})5\mathrm{mL}$ ，涡旋混匀 $30~\mathrm{s}$ 后，加入 $10~\mathrm{mL}$ 正己烷，涡旋混匀 $30~\mathrm{s}$ ，振荡提取 $15\mathrm{min}$ 后，以 $4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，将正己烷层转移至 $25~\mathrm{mL}$ 容量瓶中，向残渣加入 $10~\mathrm{mL}$ 正己烷，重复提取1次，以 $4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，合并正己烷层于同一容量瓶中，加正己烷定容至刻度，混匀。吸取 $5.0~\mathrm{mL}$ 正己烷提取液，于 $30^{\circ}\mathrm{C}$ 下氮气浓缩至干，迅速加入含抗坏血酸的乙腈溶液 $5.0~\mathrm{mL}$ ，涡旋混匀 $30~\mathrm{s}$ ，经 $0.45\mu \mathrm{m}$ 有机微孔滤膜过滤，制得试样待测液。

# 5.3 液相色谱参考条件

a）色谱柱： $\mathrm{C_{18}}$ 色谱柱 $(4.6\mathrm{mm}\times 250\mathrm{mm},5\mu \mathrm{m})$ 或性能相当者。

b）流动相：乙腈-水（ $65:35$ ，体积比）。

c) 流速: $1 \mathrm{~mL} / \mathrm{min}$ 。

d) 柱温: $30^{\circ} \mathrm{C}$ 。

e）激发波长： $365~\mathrm{nm}$ ；发射波长： $435~\mathrm{nm}$ 

f）进样体积： $10~\mu \mathrm{L}$ 

# 5.4 标准曲线的制作

将标准系列工作液分别注入液相色谱仪中，测得相应的峰面积，以标准系列工作液中乙氧基喹的浓

度为横坐标，以峰面积为纵坐标，绘制标准曲线。乙氧基喹标准溶液的液相色谱图参见附录A中图A.1。

# 5.5 试样溶液的测定

将试样待测液注入液相色谱仪中，以保留时间定性，测得相应的峰面积，根据标准曲线得到试样待测液中乙氧基喹的浓度。试样溶液中乙氧基喹色谱峰的保留时间与相应标准色谱峰的保留时间比较，变化范围应在 $\pm 2.5\%$ 内。

# 6 分析结果的表述

试样中乙氧基喹的含量按式(1)计算。

$$
X = \frac {\rho \times V _ {3} \times V _ {1} \times 1 0 0 0}{V _ {2} \times m \times 1 0 0 0 \times 1 0 0 0} \times 1. 0 2 \quad \dots \tag {1}
$$

式中：

$X$ ——试样中乙氧基喹的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho$ ——由标准曲线得到的试样溶液中乙氧基喹的浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V_{3}$ ——试样溶液的体积，单位为毫升 $(\mathrm{mL})$ 

$V_{1}$ ——提取溶液的定容体积，单位为毫升 $(\mathrm{mL})$ 

$V_{2}$ ——提取溶液分取体积，单位为毫升 $(\mathrm{mL})$ ；

1000——单位换算系数；

$m$ ——试样的取样量，单位为克 $(\mathrm{g})$ ；

1.02 ——试样制备过程中添加抗坏血酸产生的换算系数。

以重复性条件下获得的两次独立测定结果的算术平均值表示，计算结果保留2位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ 。

# 8 其他

本方法的检出限为 $0.01\mathrm{mg / kg}$ ，定量限为 $0.02\mathrm{mg / kg}$ 

# 第二法 气相色谱法

# 9 原理

试样中的乙氧基喹在碱性条件下经正己烷提取，氮吹至干后，正己烷复溶，用配有氮磷检测器的气相色谱仪测定，外标法定量。

# 10 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 10.1 试剂

10.1.1 氢氧化钠(NaOH)。

10.1.2 抗坏血酸 $\mathrm{(C_6H_8O_6)}$ 。

10.1.3 正己烷 $(\mathrm{C_6H_{14}})$ 。

# 10.2 试剂配制

氢氧化钠溶液 $(0.1\mathrm{mol} / \mathrm{L})$ ：称取 $4\mathrm{g}$ 氢氧化钠，溶于水并稀释至 $1000~\mathrm{mL}$ 。

# 10.3 标准品

同3.3。

# 10.4 标准溶液配制

10.4.1 乙氧基喹标准储备液 $(1.00\mathrm{mg / mL})$ ：称取乙氧基喹标准品 $100\mathrm{mg}$ （精确至 $0.1\mathrm{mg}$ )于 $100~\mathrm{mL}$ 容量瓶中，用正己烷溶解并定容至刻度，混匀。将溶液转移至棕色玻璃容器中，于一 $18^{\circ}C$ 下避光保存，有效期为1个月。

10.4.2 乙氧基喹标准中间液 $(10.0\mu \mathrm{g / mL})$ ：吸取标准储备液 $(1.00\mathrm{mg / mL})1.00\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，用正己烷定容至刻度，混匀。将溶液转移至棕色玻璃容器中，于一 $18^{\circ}C$ 下避光保存，有效期为1个月。

10.4.3 乙氧基喹标准系列工作液：分别吸取标准中间液 $(10.0\mu \mathrm{g / mL})20.0\mu \mathrm{L},50.0\mu \mathrm{L},100\mu \mathrm{L}$ $200~\mu \mathrm{L},400~\mu \mathrm{L}$ 于 $10~\mathrm{mL}$ 容量瓶中，用正己烷定容至刻度，混匀。乙氧基喹标准系列工作液的浓度分别为 $20.0\mathrm{ng / mL},50.0\mathrm{ng / mL},100\mathrm{ng / mL},200\mathrm{ng / mL},400\mathrm{ng / mL}$ 。临用现配。

# 11 仪器和设备

11.1 气相色谱仪：配有氮磷检测器。

11.2 组织捣碎机。

11.3 分析天平：感量分别为 $0.01\mathrm{g}$ 和 $0.1\mathrm{mg}$ 

11.4 涡旋混合器。

11.5 振荡器。

11.6 氮气浓缩装置。

11.7 离心机。

# 12 分析步骤

# 12.1 试样制备

同5.1。

# 12.2 试样提取

称取 $5\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 试样于 $50~\mathrm{mL}$ 离心管中，加入氢氧化钠溶液 $(0.1\mathrm{mol} / \mathrm{L})5\mathrm{mL}$ ，涡旋混匀 $30~\mathrm{s}$ 后，加入 $10~\mathrm{mL}$ 正己烷，涡旋混匀 $30~\mathrm{s}$ ，振荡提取 $15\mathrm{min}$ 后，以 $4000~\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，将正己烷层转移至 $25~\mathrm{mL}$ 容量瓶中，向残渣加入 $10~\mathrm{mL}$ 正己烷，重复提取1次，以 $4000~\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，合并正己烷层于同一容量瓶中，加正己烷定容至刻度，混匀。吸取 $5.0~\mathrm{mL}$ 正己烷提取液，于 $30^{\circ}\mathrm{C}$ 下氮气浓缩

至于，迅速加入 $1.0\mathrm{mL}$ 正己烷，涡旋混匀 $30~\mathrm{s}$ ，经 $0.45\mu \mathrm{m}$ 有机微孔滤膜过滤，制得试样待测液。

# 12.3 气相色谱参考条件

气相色谱参考条件如下：

a) 色谱柱: 弱极性石英毛细管柱 (内涂 $5 \%$ 苯基甲基聚硅氧烷, $30 \mathrm{~m} \times 0.32 \mathrm{~mm} \times 0.25 \mu \mathrm{m}$ ) 或等效柱。

b）载气：氮气，流速为 $1.5\mathrm{mL / min}$ 

c）进样口温度： $250~\mathrm{^\circ C}$ □

d）检测器：温度 $300^{\circ}\mathrm{C}$ ，空气流量 $120~\mathrm{mL / min}$ ，氢气流量 $3\mathrm{mL} / \mathrm{min}$ ，尾吹气流量 $3.5  \mathrm{mL / min}$ 。

e）柱温程序：初始温度 $80^{\circ}\mathrm{C}$ ，保持 $1\mathrm{min}$ ，以 $10^{\circ}\mathrm{C / min}$ 的速率升温至 $130^{\circ}\mathrm{C}$ ，以 $5^{\circ}\mathrm{C / min}$ 的速率升温至 $180^{\circ}\mathrm{C}$ ，以 $30^{\circ}\mathrm{C / min}$ 的速率升温至 $300^{\circ}\mathrm{C}$ ，保持 $3\mathrm{min}$ 。

f）进样方式：不分流进样。

g）进样体积： $2\mu \mathrm{L}$ 

# 12.4 标准曲线的制作

将标准系列工作液分别注入气相色谱仪中，测得相应的峰面积，以标准系列工作液中乙氧基喹的浓度为横坐标，以峰面积为纵坐标，绘制标准曲线。标准溶液的气相色谱图参见附录A中图A.2。

# 12.5 试样溶液的测定

将试样待测液注入气相色谱仪中，以保留时间定性，测得相应的峰面积，根据标准曲线得到试样待测液中乙氧基喹的浓度。试样溶液中乙氧基喹色谱峰的保留时间与相应标准色谱峰的保留时间比较，变化范围应在 $\pm 0.5\%$ 内。

$$

式中：

$X$ ——试样中乙氧基喹的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho$ ——由标准曲线得到的试样溶液中乙氧基喹的浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V_{3}$ ——试样溶液的体积，单位为毫升 $(\mathrm{mL})$
