"## GB 2760[—] 2024 \n\n- 3.2[在下列情况下可使用食品添加剂][:] \n\n   - a[)][保持或提高食品本身的营养价值][;] \n\n   - b[)][ 作为某些特殊膳食用食品的必要配料或成分][;] \n\n   - c[)][ 提高食品的质量和稳定性][,][改进其感官特性][;] \n\n   - d[)][ 便于食品的生产][、][加工][、][包装][、][运输或者贮藏][。] \n\n- 3.3[食品添加剂质量标准] 按照本标准使用的食品添加剂应当符合相应的质量规格要求。 \n\n- 3.4 \n\n- .44[带入原则] \n\n- 3.4.1[在下列情况下食品添加剂可以通过食品配料][(][含食品添加剂][)][带入食品中][:] \n\n   - a[)][根据本标准][,][食品配料中允许使用该食品添加剂][;] \n\n   - b[)][ 食品配料中该添加剂的用量不应超过允许的最大使用量][;] \n\n   - c[)][ 应在正常生产工艺条件下使用这些配料][,][并且食品中该添加剂的含量不应超过由配料带入的] 水平; \n\n   - d[)][ 由配料带入食品中的该添加剂的含量应明显低于直接将其添加到该食品中通常所需要的] 水平。 \n\n3.4.2[当某食品配料作为特定终产品的原料时][,][批准用于上述特定终产品的添加剂允许添加到这些食] 品配料中,同时该添加剂在终产品中的量应符合本标准的要求。在所述特定食品配料的标签上应明确 标示该食品配料用于上述特定食品的生产。 \n\n- 4[食品分类系统] \n\n食品分类系统用于界定食品添加剂的使用范围,只适用于本标准,见附录 E。 \n\n- 5[食品添加剂的使用规定] \n\n食品添加剂的使用应符合附录 A 的规定。 \n\n- 6[食品用香料] \n\n用于生产食品用香精的食品用香料的使用应符合附录 B 的规定。 \n\n7[食品工业用加工助剂] \n\n食品工业用加工助剂的使用应符合附录 C 的规定。 \n\n- 8[食品添加剂的功能类别] \n\n食品添加剂的功能类别见附录 D。 \n\n- 9[附录] A[中食品添加剂使用规定索引] \n\n附录 A 中食品添加剂使用规定索引见附录 F。 \n\n"
