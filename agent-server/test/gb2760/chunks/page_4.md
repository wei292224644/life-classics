"# 食品安全国家标准 \n\n# 食品添加剂使用标准 \n\n- 1[范围] \n\n本标准规定了食品添加剂的使用原则、允许使用的食品添加剂品种、使用范围及最大使用量或残 留量。 \n\n- 2[术语和定义] \n\n- 2.1 \n\n- .11[食品添加剂] \n\n为改善食品品质和色、香、味,以及为防腐、保鲜和加工工艺的需要而加入食品中的人工合成或者天 然物质。食品用香料、胶基糖果中基础剂物质、食品工业用加工助剂、营养强化剂也包括在内。 \n\n- 2.2[最大使用量] \n\n食品添加剂使用时所允许的最大添加量。 \n\n- 2.3[最大残留量] \n\n食品添加剂或其分解产物在最终食品中的允许残留水平。 \n\n- 2.4[食品工业用加工助剂] \n\n保证食品加工能顺利进行的各种物质,与食品本身无关。如助滤、澄清、吸附、脱模、脱色、脱皮、提 取溶剂、发酵用营养物质等。 \n\n- 2.5[食品添加剂编码] \n\n用于代替复杂的化学结构名称表述的编码,包括食品添加剂的国际编码系统(INS)a 和中国编码系 \n\n统 (CNS)。 \n\n- 3[食品添加剂的使用原则] \n\n- 3.1[食品添加剂使用时应符合以下基本要求][:] \n\n   - a[)][不应对人体产生任何健康危害][;] \n\n   - b[)][ 不应掩盖食品腐败变质][;] \n\n   - c[)][ 不应掩盖食品本身或加工过程中的质量缺陷或以掺杂][、][掺假][、][伪造为目的而使用食品添加剂][;] \n\n   - d[)][ 不应降低食品本身的营养价值][;] \n\n   - e[)][ 在达到预期效果的前提下尽可能降低在食品中的使用量][。] \n\n   - aINS号是国际食品法典委员会 (CAC)赋予食品添加剂的编号系统,当 CAC 的 INS 号发生变化时以 CAC 的 INS[号为准][。] \n\n"
