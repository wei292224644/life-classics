📄 product/index.vue — 页面布局
#1

Banner 高度过大

v14 设计稿 banner 高度为 260px（在 375px 宽手机上），当前实现为 320px，视觉上明显偏高。

v14height: 260px → 520rpx
现在height: 320px（偏大 23%）
#2

缺少"营养成分"章节标题

v14 在 NutritionCard 前有 <h3> 营养成分 标题，当前页面没有。

v14<h3 class="section-title">营养成分</h3>
现在直接渲染 <NutritionCard>，无标题
#3

内容区 padding-top 缺失

v14 content 有 24px 顶部 padding，现在为 0，导致内容紧贴 banner。

v14padding: 24px 20px 40px → 48rpx 40rpx
现在padding: 0 40rpx 200rpx
🖼 product-banner — Banner 区域
#4

占位 Emoji 尺寸过大

v14 emoji 为 80px，当前 120px，在 banner 中过于突出。

v14font-size: 80px → 160rpx
现在font-size: 120px（偏大 50%）
#5

Banner 缺少"产品图片"副标题文字

v14 在 emoji 下方有 banner-label 文字"产品图片"，当前没有该文字。

v14<div class="banner-label">产品图片</div>
现在无此文字
🃏 RiskGroup.vue — 风险分组
#6

全部使用 px 单位（无响应式缩放）

border-radius、padding、margin、font-size 全部用 px，不随屏幕缩放。

v1420px → 40rpx, 16px → 32rpx
现在border-radius: 20px; padding: 16px;
💚 HealthBenefitCard.vue + AiAdviceCard.vue
#7

HealthBenefitCard 内嵌了"健康益处"标题

v14 中章节标题在卡片外，由页面统一管理。当前 HealthBenefitCard 内含 section-title，导致与其他章节样式不一致。

v14<h3>健康益处</h3> 在 health-card 外
现在section-title 在 HealthBenefitCard 内部
#8

全部使用 px 单位 + 卡片背景色不对（暗色）

v14 暗色模式健康/建议卡背景是 rgba(255,255,255,0.02)（近透明），当前使用 var(--bg-card) = #1a1a1a（不透明深色），视觉差异明显。

v14bg: rgba(255,255,255,0.02), border: rgba(255,255,255,0.06)
现在bg: var(--bg-card) = #1a1a1a（太深）
🔘 BottomBar.vue — 底部按钮栏
#9

次要按钮背景透明（应有淡白底）

v14 暗色模式"添加到记录"按钮背景为 rgba(255,255,255,0.06)，当前为 transparent，有轻微区别。

v14background: rgba(255,255,255,0.06)
现在background: transparent
🔝 ProductHeader.vue — 顶部导航
#10

图标颜色写死为 #fff，不跟随滚动状态变化

v14 在 scrolled 状态下，图标应变为 var(--text-primary)（亮色模式为深色）。当前图标颜色始终为 #fff。

v14scrolled 后 icon color: var(--header-text)
现在固定 color="#fff"
