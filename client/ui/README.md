# AI 健康助手界面设计方案

## 📋 设计概览

本设计方案为健康管理 App 的智能客服 / AI 健康助手界面提供完整的 UI/UX 设计规范与组件实现。

## 🎨 设计原则

- **健康感**：色彩柔和、自然，传达健康与信任
- **易用性**：清晰的信息层级，直观的交互流程
- **现代感**：大圆角、模块化卡片、适度留白
- **情感化**：温和的动效，友好的视觉反馈

---

## 📐 页面结构树（Component Tree）

```
HealthChatPage (主页面)
├── ChatHeader (顶部栏)
│   ├── Avatar (AI助手头像)
│   ├── Title (标题)
│   └── SettingsButton (设置入口)
│
├── ChatContainer (对话容器)
│   ├── MessageList (消息列表)
│   │   ├── UserMessage (用户消息)
│   │   │   ├── MessageBubble
│   │   │   └── ImagePreview (可选)
│   │   │
│   │   └── AIMessage (AI消息)
│   │       ├── MessageBubble
│   │       ├── NutritionCard (营养卡片)
│   │       ├── RiskTag (风险标签)
│   │       ├── SuggestionCard (建议模块)
│   │       ├── ReasoningIndicator (推理过程指示器，可选)
│   │       └── ActionButtons (操作按钮：重新提问/生成总结/生成建议)
│   │
│   └── QuickActions (快捷提问按钮)
│       └── QuickActionButton
│
└── InputBar (底部输入栏)
    ├── TextInput (文本输入框)
    ├── VoiceButton (语音输入按钮)
    ├── CameraButton (相机按钮)
    └── GalleryButton (图库按钮)
```

## 🎨 颜色方案

### Primary（主色调）
- **主色**: `#4A90E2` (健康蓝，信任感)
- **主色浅**: `#E8F2FC` (背景/高亮)
- **主色深**: `#357ABD` (悬停状态)

### Secondary（辅助色）
- **成功绿**: `#52C41A` (健康建议/正面信息)
- **警告橙**: `#FA8C16` (中等风险)
- **危险红**: `#F5222D` (高风险/警告)
- **信息灰**: `#8C8C8C` (次要信息)

### Background（背景色）
- **主背景**: `#FAFAFA` (浅灰白，护眼)
- **卡片背景**: `#FFFFFF` (纯白)
- **输入背景**: `#F5F5F5` (浅灰)
- **分隔线**: `#E8E8E8` (淡灰)

### Text（文字色）
- **主文字**: `#262626` (深灰，高对比度)
- **次要文字**: `#8C8C8C` (中灰)
- **占位文字**: `#BFBFBF` (浅灰)

## 🎯 组件说明

### 1. ChatHeader（顶部栏）
- **作用**: 显示AI助手身份、页面标题、设置入口
- **布局**: 左对齐头像+标题，右对齐设置按钮
- **高度**: `64px`

### 2. MessageList（消息列表）
- **作用**: 展示对话历史，支持滚动
- **布局**: 垂直堆叠，用户消息右对齐，AI消息左对齐
- **间距**: 消息之间 `16px`

### 3. MessageBubble（消息气泡）
- **作用**: 承载消息内容（文字、图片）
- **样式**: 用户消息（蓝色背景），AI消息（白色背景+边框）
- **圆角**: `16px` (顶部两个角为 `20px`)

### 4. NutritionCard（营养卡片）
- **作用**: 结构化展示食品营养成分
- **布局**: 网格布局，显示卡路里、蛋白质、脂肪、碳水等
- **样式**: 浅色背景，带图标

### 5. RiskTag（风险标签）
- **作用**: 标识健康风险等级
- **类型**: 低风险（绿）、中风险（橙）、高风险（红）
- **样式**: 圆角标签，带图标

### 6. SuggestionCard（建议模块）
- **作用**: 展示AI的健康建议
- **布局**: 列表式，带图标和标题
- **样式**: 卡片式，可展开/收起

### 7. QuickActions（快捷提问）
- **作用**: 提供常用问题快捷入口
- **布局**: 横向滚动，按钮式
- **样式**: 圆角按钮，浅色背景

### 8. InputBar（输入栏）
- **作用**: 文本输入、语音、图片上传入口
- **布局**: 输入框居中，功能按钮两侧
- **高度**: `56px` (含安全区域)

## 🎬 交互说明

### 输入交互
- **文本输入**: 聚焦时输入框边框高亮（主色），占位文字淡出
- **发送消息**: 点击发送按钮或按 Enter，消息渐入动画（`fadeIn + slideUp`）
- **语音输入**: 长按按钮开始录音，松开发送，显示波形动画

### 滚动交互
- **消息列表**: 新消息自动滚动到底部，平滑滚动
- **快捷按钮**: 横向滚动，支持左右滑动

### 上传图片
- **点击相机/图库**: 打开系统选择器
- **预览**: 上传后显示缩略图，可删除
- **加载状态**: 显示进度条或骨架屏

### 长按交互
- **消息长按**: 显示操作菜单（复制、删除、重新提问）
- **图片长按**: 显示预览/删除选项

### 动效规范
- **渐入**: `opacity: 0 → 1`, `duration: 300ms`, `ease-out`
- **滑入**: `translateY: 10px → 0`, `duration: 300ms`, `ease-out`
- **缩放**: `scale: 0.95 → 1`, `duration: 200ms`, `ease-out`
- **悬停**: `scale: 1 → 1.02`, `duration: 150ms`



## 📱 响应式设计

- **移动端优先**: 默认适配手机屏幕（375px - 768px）
- **平板适配**: 768px+ 时调整布局，增加卡片宽度
- **安全区域**: 底部输入栏考虑 iPhone 底部安全区域
