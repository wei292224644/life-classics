# 食品安全国家标准

# 食品添加剂 十四酸乙酯(肉豆蔻酸乙酯)

# 1 范围

本标准适用于由十四酸和乙醇为原料制得的食品添加剂十四酸乙酯(肉豆蔻酸乙酯)。

# 2 化学名称、分子式、结构式和相对分子质量

# 2.1 化学名称

十四酸乙酯

# 2.2 分子式

$$
\mathrm {C} _ {1 6} \mathrm {H} _ {3 2} \mathrm {O} _ {2}
$$

# 2.3 结构式

$$
\mathrm {C H} _ {3} \left(\mathrm {C H} _ {2}\right) _ {1 2} \mathrm {C O O C H} _ {2} \mathrm {C H} _ {3}
$$

# 2.4 相对分子质量

256.43(按 2007 年国际相对原子质量)

# B.2 操作条件

B.2.1 柱：毛细管柱，长 $3 0 \mathrm { m }$ ，内径 $0 . 3 2 \mathrm { m m }$ 。

B.2.2 固定相： $5 \%$ 苯基)甲基聚硅氧烷。

B.2.3 膜厚： $0 . 2 5 \mu \mathrm { m }$ 。

B.2.4 色谱炉温度： $2 0 0 ^ { \circ } \mathrm { C }$ 恒温 $3 \mathrm { m i n }$ ，然后线性程序升温从 $2 0 0 ^ { \circ } \mathrm { C } { \sim } 2 6 0 ^ { \circ } \mathrm { C }$ ，速率 $8 \mathrm { ^ { \circ } C / m i n }$ ，最后在 $2 6 0 \mathrm { { ^ \circ C } }$ 恒温 $1 8 \mathrm { { m i n } }$ 。

B.2.5 进样口温度： $2 5 0 \mathrm { { ^ \circ C } }$ 。

B.2.6 检测器温度： $3 0 0 \mathrm { { ^ \circ C } }$ 。

B.2.7 检测器：氢火焰离子化检测器。

B.2.8 载气：氮气。

B.2.9 柱前压：90kPa。

B.2.10 进样量： $0 . 2 \mu \mathrm { L }$ 。

B.2.11 分流比：1/50。
