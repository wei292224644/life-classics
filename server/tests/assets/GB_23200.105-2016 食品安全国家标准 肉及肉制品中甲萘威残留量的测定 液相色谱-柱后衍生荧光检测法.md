# 前言

本标准按照GB/T1.1—2009给出的规则起草。

本标准代替SN/T0122—2011《进出口肉及肉制品中甲萘威残留量检验方法液相色谱一柱后衍生荧光检测法》。与SN/T0122—2011相比，主要变化如下：

标准文本格式修改为食品安全国家标准文本格式；

——标准名称中“进出口肉及肉制品”改为“肉及肉制品”；

——标准范围中增加“其他食品可参照执行”。

# 1 范围

本标准规定了肉及肉制品中甲萘威残留量的液相色谱一柱后衍生荧光检测法。

本标准适用于牛肉、鸡肉、虾肉、鱼肉及火腿罐头中甲萘威残留量的测定，其他食品可参照执行。

# 2 规范性引用文件

下列文件对于本文件的应用是必不可少的。凡是注日期的引用文件，仅注日期的版本适用于本文件。凡是不注日期的引用文件，其最新版本(包括所有的修改单)适用于本文件。

# 3 原理

用丙酮一石油醚混合溶液提取样品中的甲萘威残留物，经凝胶层析柱净化后，浓缩，高效液相色谱分离，经柱后衍生后，用荧光检测器检测，外标法定量。

# 4 试剂和材料

除另有规定外，所有试剂均为分析纯试剂，水为符合GB/T6682中规定的一级水。

# 4.1 试剂

4.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
4.1.2 乙酸乙酯 $\mathrm{(C_4H_8O_2)}$ ：色谱纯。  
4.1.3 环己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{12}\right)$ ：色谱纯。  
4.1.4 丙酮 $\mathrm{(CH_3COCH_3)}$ ：色谱纯。  
4.1.5 石油醚：沸程 $30^{\circ}\mathrm{C}\sim 60^{\circ}\mathrm{C}$   
4.1.6 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ ： $650^{\circ}\mathrm{C}$ 灼烧 $4\mathrm{h}$ ，在干燥器内冷却至室温，储于密封瓶中备用。  
4.1.7 氯化钠(NaCl)。  
4.1.8 柱后衍生试剂。  
4.1.9 邻苯二甲醛 $\mathrm{(C_8H_6O_2}$ ，O-Phthalaldehyde，OPA）。  
4.1.10 疏基乙醇 $\mathrm{(C_2H_6OS}$ ，Thiofluor）。

# 4.2 溶液配制

4.2.1 NaOH溶液 $(0.2\% ,m / v)$ ：取 $2\mathrm{gNaOH}$ ，以水定容至 $1000~\mathrm{mL}$ 。  
4.2.2 邻苯二甲醛稀释液：硼砂溶液 $(0.4\% ,m / v)$ 。  
4.2.3 邻苯二甲醛试液：溶剂储罐中注入 $945\mathrm{mL}$ OPA 稀释剂，用惰性气体（氮气）吹扫至少 $10\mathrm{min}$ ， $100\mathrm{mg}$ OPA 固体溶解于约 $10\mathrm{mL}$ 的色谱纯甲醇中，将 OPA 溶液加入除氧的 OPA 稀释剂中，溶解 $2\mathrm{g}$ 硫基乙醇固体于 $5\mathrm{mL}$ OPA 稀释剂中，加入到储罐中，盖上瓶盖，打开气流，再不断地吹扫几分钟，关闭

排气阀，轻轻地搅动溶剂以使其完全混合。

4.2.4 乙酸乙酯一环己烷混合溶液(1+1，体积比)：取 $100\mathrm{mL}$ 乙酸乙酯，加入 $100\mathrm{mL}$ 环己烷，摇匀备用。

4.3 标准品

甲萘威：分子式 $\mathrm{C_{12}H_{11}NO_2}$ ，CAS号：63-25-2，纯度 $>99.5\%$

4.4 标准溶液配制

4.4.1 甲萘威标准储备液

准确称取适量的甲萘威标准品，以乙腈溶解， $4^{\circ}\mathrm{C}$ 冰箱保存，有效期为6个月。保存于 $4^{\circ}\mathrm{C}$ 冰箱内。

4.4.2 甲萘威标准工作液

取一定量的标准储备液, 用乙腈稀释至适当浓度, $4^{\circ} \mathrm{C}$ 冰箱保存, 有效期为 1 周。保存于 $4^{\circ} \mathrm{C}$ 冰箱内。

4.5 材料

$0.45\mu \mathrm{m}$ 尼龙滤膜。

5 仪器和设备

5.1 液相色谱仪：配有荧光检测器和柱后衍生单元。  
5.2 分析天平：感量 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$   
5.3全自动凝胶色谱仪（配有馏分收集浓缩器）。  
5.4 旋转蒸发装置。  
5.5 氮吹仪。  
5.6 组织匀浆机。  
5.7 涡旋振荡器。

6 试样制备与保存

6.1 试样部位

样品取样部位按GB2763的规定执行。在抽样和制样的操作中，应防止样品受到污染或发生含量的变化，以保证实验样品能代表总体样本。

6.2 试样制备

6.2.1 肉

将所取全部样品充分搅碎混匀, 取有代表性的样品, 总量不少于 $500 \mathrm{~g}$ , 装入清洁容器内, 密封并标明标记。

6.2.2 罐头

将所取全部样品整罐倒出，充分搅碎混匀，取有代表性的样品，总量不少于 $500\mathrm{g}$ ，装入清洁容器密封并标明标记。

6.3 试样保存

试样应于 $-18^{\circ}\mathrm{C}$ 以下保存。

7 分析步骤

7.1 提取

称取试样 $20\mathrm{g}$ (精确到 $0.01\mathrm{g}$ ), 于 $100\mathrm{mL}$ 具塞三角瓶中, 加水 $6\mathrm{mL}$ (视样品水分含量加水使总水量约 $20\mathrm{g}$ , 肉通常在 $70\%$ 左右, 加水 $6\mathrm{mL}$ ), 加 $40\mathrm{mL}$ 丙酮, 匀浆 $1\mathrm{min}$ , 加氯化钠 $6\mathrm{g}$ , 充分摇匀, 再加

$30~\mathrm{mL}$ 石油醚，振摇 $30\mathrm{min}$ 。取 $35\mathrm{mL}$ 有机层上清液，经无水硫酸钠滤于旋转蒸发瓶中，浓缩至约 $1\mathrm{mL}$ 加 $2\mathrm{mL}$ 乙酸乙酯一环己烷溶液再浓缩，如此重复3次。乙酸乙酯一环己烷定容为 $5\mathrm{mL},0.45\mu \mathrm{m}$ 滤膜过滤，待净化。

# 7.2 凝胶色谱(GPC)净化

# 7.2.1 凝胶色谱净化参考条件

a）净化柱： $400\mathrm{mm}\times 30\mathrm{mm}$ ,Bio Beads S-X3，或相当者；  
b）流动相：乙酸乙酯一环己烷；  
c) 流速: $5.0 \mathrm{~mL} / \mathrm{min}$ ;   
d) 样品定量环: $5 \mathrm{~mL}$ ;   
e) 馏分收集段： $10.0\mathrm{min} \sim 15.0\mathrm{min}$ ；  
f）净化柱平衡时间： $5\mathrm{min}$

# 7.2.2 凝胶色谱浓缩参考条件

a）样品浓缩条件见表1；  
b）浓缩终点判定：液位传感模式；  
c) 溶剂替换: $2 \mathrm{~mL}$ 乙腈, 重复 2 次;  
d） 定容体积： $1\mathrm{mL}$

# 7.2.3 凝胶色谱净化步骤

将 $5\mathrm{mL}$ 待净化液按7.2.1和7.2.2规定的条件进行净化与浓缩，最后得到 $1\mathrm{mL}$ 净化液待测定。

# 7.3 测定

# 7.3.1 液相色谱参考条件

a）色谱柱： $\mathrm{C_{18}}$ ， $250\mathrm{mm}\times 4.6\mathrm{mm}$ ，膜厚 $5\mu \mathrm{m}$ ，或相当者；  
b) 柱温: $42^{\circ} \mathrm{C}$ ;   
c）荧光检测器： $\lambda_{\mathrm{ex}}330\mathrm{nm},\lambda_{\mathrm{em}}465\mathrm{nm}$   
d) 流动相：乙腈 $+$ 水 $(40 + 60$ ，体积比）；  
e) 流速: $1.0 \mathrm{~mL} / \mathrm{min}$ ;   
f）进样量： $20\mathrm{L}$

# 7.3.2 柱后衍生

a) 衍生试剂 1: 水解试剂, $0.4\%$ 氢氧化钠溶液, 流速 $0.4 \mathrm{~mL} / \mathrm{min}$ ;  
b）衍生试剂2：OPA试剂，流速 $0.4\mathrm{mL / min}$   
c）反应器温度：水解温度为 $100^{\circ}\mathrm{C}$ ，衍生温度为室温。

# 7.3.3 色谱测定

根据样液中被测甲萘威的含量情况，选定浓度相近的标准工作液，其响应值应在方法检测的线性范围内。在上述液相色谱条件下，甲萘威保留时间为 $11.1\mathrm{min}$ ，色谱图参见图A.1。

# 7.4 空白试验

除不加试样外，均按 $7.1\sim 7.3$ 的规定执行。

# 8 结果计算和表述

按式(1)计算试样中甲萘威的含量。

$$
X = \frac {A \times C _ {s} \times V}{A _ {s} \times m}. \tag {1}
$$

式中：

$X$ ——试样中甲萘威的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$A$ ——试样中甲萘威的色谱峰面积；

$C_s$ ——标准工作溶液中甲萘威的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——样液最终定容体积,单位为毫升 $(\mathrm{mL})_{\circ}$

$A_{s}$ ——标准工作溶液中甲萘威的色谱峰面积；

$m$ ——最终样液所代表的量，单位为克 $(\mathrm{g})$ 。

计算结果须扣除空白值，测定结果用平行测定的算术平均值表示，保留2位有效数字。

# 9 精密度

9.1 在重复性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值 (百分率), 应符合附录 B 的要求。  
9.2 在再现性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值(百分率), 应符合附录 C 的要求。
