# 前言

本标准的附录 A 为资料性附录。

本标准系国内首次发布的国家标准。

# 1 范围

本标准规定了牛奶中甲砜霉素残留量检测的制样和高效液相色谱测定方法。

本标准适用于牛奶中甲砜霉素残留量的检测。

# 3 原理

试料中残留的甲砜霉素，用乙酸乙酯提取，正己烷除脂， $\mathrm{C}_{18}$ 柱净化，高效液相色谱 - 紫外测定，外标法定量。

# 4 试剂和材料

以下所用的试剂，除特别注明外均为分析纯，水为符合 GB/T6682 规定的一级水。

4.1 甲砜霉素对照品：含量 $\geqslant 99.0\%$   
4.2 乙酸乙酯  
4.3 正己烷  
4.4 乙腈：色谱纯。  
4.5 $\mathrm{C_{18}}$ 固相萃取柱： $200\mathrm{mg / 3mL}$ ，或相当者。  
4.6 $1 \mathrm{mg} / \mathrm{mL}$ 甲砜霉素标准贮备液：精密称取甲砜霉素对照品 $10 \mathrm{mg}$ ，于 $10 \mathrm{~mL}$ 量瓶中，用乙腈溶解并稀释至刻度，配制成浓度为 $1 \mathrm{mg} / \mathrm{mL}$ 的标准贮备液。 $2 \sim 8^{\circ} \mathrm{C}$ 保存，有效期 1 个月。  
4.7 $10\mu \mathrm{g / mL}$ 甲砜霉素标准工作液：精密量取 $1\mathrm{mg / mL}$ 甲砜霉素标准贮备液 $1.0~\mathrm{mL}$ ，于 $100~\mathrm{mL}$ 量瓶

中，用流动相稀释，配制成浓度为 $10\mu \mathrm{g / mL}$ 的甲砜霉素标准工作液。现配现用。

# 5 仪器和设备

5.1 高效液相色谱仪：配紫外检测器。  
5.2 分析天平：感量 $0.00001\mathrm{g}$ 。  
5.3 天平：感量 $0.01\mathrm{g}$ 。  
5.4 离心机  
5.5 固相萃取装置。  
5.6 氮气吹干仪。  
5.7 振荡混合器。  
5.8 鸡心瓶： $150 \mathrm{ml}$ 。  
5.9 旋转蒸发仪。  
5.10 具塞塑料离心管： $50~\mathrm{mL}$ 。  
5.11 滤膜： $0.45 \mu \mathrm{m}$ 。

# 6 试料的制备与保存

## 6.1 试料的制备

取适量新鲜或冷藏的空白或供试牛奶，混合均匀。

——取均质后的供试样品，作为供试试料。  
——取均质后的空白样品，作为空白试料。  
——取均质后的空白样品，添加适宜浓度的标准溶液，作为空白添加试料。

## 6.2 试料的保存

-20°C 以下保存。

# 7 测定步骤

## 7.1 提取

称取试料（ $5 \pm 0.05$ ）g，于 $50 \mathrm{~mL}$ 离心管中，加乙酸乙酯 $20 \mathrm{~mL}$ ，振荡 $10 \mathrm{~min}$ ， $4000 \mathrm{r} / \mathrm{min}$ 离心 $5 \mathrm{~min}$ ，取上清液于鸡心瓶中，残渣中加乙酸乙酯 $20 \mathrm{~mL}$ ，重复提取一次，合并两次提取液，于 $45^{\circ} \mathrm{C}$ 旋

转蒸发至近干，加水 $5 \mathrm{~mL}$ 于鸡心瓶中，超声 $5 \mathrm{~min}$ 使充分溶解，转至 $50 \mathrm{~mL}$ 离心管中。再加水 $5 \mathrm{~mL}$ 于鸡心瓶中重复溶解后转至同一离心管中。加 $20 \mathrm{~mL}$ 正己烷，振荡 $5 \mathrm{~min}$ ， $4000 \mathrm{r} / \mathrm{min}$ 离心 $2 \mathrm{~min}$ ，取下层液备用。

## 7.2 净化

$\mathrm{C_{18}}$ 柱依次用乙腈 $5\mathrm{mL}$ 和水 $5\mathrm{mL}$ 活化。取备用液过柱，控制流速 $1\mathrm{mL / min}$ ，挤干。用乙腈 $5\mathrm{mL}$ 洗脱，收集洗脱液，于 $45^{\circ}\mathrm{C}$ 氮气吹干，用流动相 $1.0\mathrm{mL}$ 溶解残余物，滤膜过滤，供高效液相色谱测定。

## 7.3.标准曲线的制备

精密量取 $10~\mu \mathrm{g / mL}$ 甲砜霉素标准工作液适量，用流动相稀释，配制成浓度为 20、50、125、250 和 $500~\mu \mathrm{g / L}$ 的系列标准溶液，供高效液相色谱测定。以测得峰面积为纵坐标，对应的标准溶液浓度为横坐标，绘制标准曲线。求回归方程和相关系数。

## 7.4 高效液相色谱测定

### 7.4.1 色谱条件

色谱柱： $\mathrm{C_{18}}$ （ $250 \mathrm{~mm} \times 4.6 \mathrm{~mm}$ ，粒径 $5 \mu \mathrm{m}$ ），或相当者；

流动相：乙腈 + 水（20+80，v/v），使用前经滤膜过滤，超声；

流速： $1.0\mathrm{mL / min}$

检测波长： $225\mathrm{nm}$

柱温： $30^{\circ}\mathrm{C}$ ；

进样量： $20~\mu \mathrm{L}$

### 7.4.2 测定法

取试样溶液和相应的标准溶液，作单点或多点校准，按外标法，以峰面积计算。标准溶液及试样溶液中甲砜霉素响应值应在仪器检测的线性范围之内。在上述色谱条件下，标准溶液和空白组织添加试样溶液的高效液相色谱图分别见附录 A。

## 7.5 空白试验

除不加试料外，采用完全相同的步骤进行平行操作。

# 8 结果计算和表述

试料中甲砜霉素的残留量（ $\mu \mathrm{g} / \mathrm{kg}$ ）：按下式计算：

$$
X = \frac {C \times V}{m}
$$

式中：

X - - - - - 供试试料中甲砜霉素的残留量，$\mu \mathrm{g} / \mathrm{kg}$ ;

C - - - - 试样溶液中甲砜霉素的浓度，ng/mL;

V - - - - - 溶解残余物流动相体积，mL；

m- - 供试试料质量，g。

注：计算结果需扣除空白值，测定结果用平行测定的算术平均值表示，保留三位有效数字。

# 9 检测方法灵敏度、准确度、精密度

## 9.1 灵敏度

本方法的检测限为 $10~\mu \mathrm{g / kg}$ ，定量限为 $10~\mu \mathrm{g / kg}$

## 9.2 准确度

本方法在 $10\sim 100~\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $70\% \sim 110\%$

## 9.3 精密度

本方法的批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 15\%$ 。
